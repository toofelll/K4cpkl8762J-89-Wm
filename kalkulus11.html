Part 11 • Persamaan Diferensial & Transformasi

📖 Pendahuluan — Matematika Perubahan

Seluruh kalkulus yang kita pelajari—dari limit sampai integral lipat tiga— bermuara di sini: persamaan diferensial. Persamaan diferensial adalah persamaan yang melibatkan fungsi yang tidak diketahui dan turunan-turunannya. Ini adalah bahasa alam semesta untuk mendeskripsikan perubahan.

Hukum Newton (F = ma) adalah persamaan diferensial. Persamaan gelombang, persamaan panas, persamaan Schrödinger dalam mekanika kuantum—semuanya persamaan diferensial. Setiap kali kamu melihat sesuatu berubah terhadap waktu dan perubahan itu bergantung pada keadaan saat ini, di situ ada persamaan diferensial.

Di bab terakhir ini, kita akan mempelajari persamaan diferensial orde satu (misalnya: dy/dx = ky untuk pertumbuhan eksponensial), persamaan diferensial orde dua (misalnya: d²y/dt² + ω²y = 0 untuk osilasi harmonik), dan dua alat transformasi paling powerful: Transformasi Laplace (mengubah PD menjadi persamaan aljabar) dan Deret Fourier (menguraikan fungsi periodik menjadi gelombang sinus dan cosinus). Kita juga akan mengupas differential forms sebagai pengantar ke geometri diferensial modern.

Tujuan Bab
  • Memahami apa itu persamaan diferensial dan mengapa ia fundamental.
  • Mampu menyelesaikan PD orde satu: separable, linear, dan eksak.
  • Menguasai PD orde dua linear homogen dengan koefisien konstan.
  • Memahami metode koefisien tak tentu dan variasi parameter.
  • Mengenal Transformasi Laplace dan penggunaannya untuk menyelesaikan PD.
  • Memahami Deret Fourier sebagai cara menguraikan sinyal periodik.
  • Mengenal differential forms sebagai bahasa geometri diferensial.

1. Persamaan Diferensial — Bahasa Perubahan

Persamaan diferensial (PD) adalah persamaan yang mengandung turunan dari fungsi yang tidak diketahui.

Orde PD: orde turunan tertinggi yang muncul.

PD Biasa (ODE): fungsi satu variabel. Contoh: dy/dx = ky.

PD Parsial (PDE): fungsi multivariabel. Contoh: ∂u/∂t = ∂²u/∂x² (persamaan panas).

Solusi PD: fungsi yang memenuhi persamaan tersebut. Solusi umum mengandung konstanta sembarang. Solusi khusus memenuhi syarat awal/batas.

Mengapa PD penting? Karena hukum alam dinyatakan dalam bentuk PD. Newton: F = ma → m(d²x/dt²) = F(x). Pertumbuhan populasi: dP/dt = kP. Pendinginan Newton: dT/dt = -k(T - T_lingkungan). Rangkaian listrik: L(d²q/dt²) + R(dq/dt) + q/C = V(t). Semuanya PD!

2. Persamaan Diferensial Orde Satu — Fondasi

2.1 PD Separable — Variabel Bisa Dipisahkan

Bentuk: dy/dx = g(x)·h(y)

Metode: Pisahkan variabel: (1/h(y)) dy = g(x) dx. Integralkan kedua ruas.
Soal: Selesaikan dy/dx = ky (pertumbuhan eksponensial).

(1/y) dy = k dx
∫ (1/y) dy = ∫ k dx
ln |y| = kx + C
y = e^(kx+C) = e^C · e^(kx) = Ae^(kx) ✅

Ini adalah model pertumbuhan populasi, peluruhan radioaktif, bunga majemuk kontinu!

2.2 PD Linear Orde Satu — Faktor Integrasi

Bentuk standar: dy/dx + P(x)y = Q(x)

Faktor integrasi: μ(x) = e^(∫P(x)dx)

Solusi: y = (1/μ(x)) ∫ μ(x)Q(x) dx

Mengapa faktor integrasi bekerja? — Trik Jenius Leibniz

Kalikan kedua ruas PD dengan μ(x): μ(dy/dx) + μPy = μQ.

Perhatikan bahwa d(μy)/dx = μ(dy/dx) + (dμ/dx)y.

Jika dμ/dx = μP, maka ruas kiri menjadi d(μy)/dx, dan PD menjadi:
d(μy)/dx = μQ → μy = ∫ μQ dx → y = (1/μ)∫ μQ dx.

Dari dμ/dx = μP → dμ/μ = P dx → ln μ = ∫P dx → μ = e^(∫P dx). ✅
Soal: Selesaikan dy/dx + 2xy = x.

P(x) = 2x. μ(x) = e^(∫2x dx) = e^(x²).

d(e^(x²)y)/dx = x·e^(x²)
e^(x²)y = ∫ x·e^(x²) dx = (1/2)e^(x²) + C
y = 1/2 + Ce^(-x²). ✅

3. Persamaan Diferensial Orde Dua — Osilasi dan Gelombang

Bentuk umum: ay'' + by' + cy = f(x)   (a, b, c konstanta, a ≠ 0).

Homogen: f(x) = 0. Non-homogen: f(x) ≠ 0.

Solusi umum = solusi homogen + solusi partikular: y = yₕ + yₚ

3.1 PD Orde Dua Homogen — Persamaan Karakteristik

Persamaan karakteristik: ar² + br + c = 0

Kasus 1 (D > 0, dua akar real berbeda r₁, r₂):
yₕ = C₁e^(r₁x) + C₂e^(r₂x)

Kasus 2 (D = 0, akar kembar r):
yₕ = C₁e^(rx) + C₂xe^(rx)

Kasus 3 (D < 0, akar kompleks α ± βi):
yₕ = e^(αx)(C₁ cos βx + C₂ sin βx)

Mengapa mencoba y = e^(rx)? — Kejeniusan Euler

Untuk PD ay'' + by' + cy = 0, coba solusi y = e^(rx).

y' = re^(rx), y'' = r²e^(rx).

Substitusi: a(r²e^(rx)) + b(re^(rx)) + c(e^(rx)) = 0
e^(rx)(ar² + br + c) = 0.

Karena e^(rx) ≠ 0, maka ar² + br + c = 0 (persamaan karakteristik). ✅

Inilah kenapa persamaan diferensial linear punya solusi eksponensial: karena turunan e^(rx) adalah kelipatan dari e^(rx) itu sendiri.
Soal: Selesaikan y'' - 5y' + 6y = 0.

Persamaan karakteristik: r² - 5r + 6 = 0 → (r-2)(r-3) = 0 → r = 2, 3.

y = C₁e^(2x) + C₂e^(3x). ✅


Soal (osilasi): Selesaikan y'' + 4y = 0 (gerak harmonik sederhana).

r² + 4 = 0 → r = ±2i. α = 0, β = 2.

y = e⁰(C₁ cos 2x + C₂ sin 2x) = C₁ cos 2x + C₂ sin 2x. ✅

Ini adalah gerak osilasi dengan frekuensi sudut ω = 2!

3.2 PD Orde Dua Non-Homogen — Mencari Solusi Partikular

Metode Koefisien Tak Tentu: Tebak bentuk yₚ berdasarkan f(x):

• f(x) = polinomial → yₚ = polinomial dengan derajat sama.
• f(x) = e^(kx) → yₚ = Ae^(kx).
• f(x) = sin ωx atau cos ωx → yₚ = A cos ωx + B sin ωx.
• f(x) = e^(kx) sin ωx → yₚ = e^(kx)(A cos ωx + B sin ωx).

Substitusi yₚ ke PD, samakan koefisien, cari A, B, C, ...
Soal: y'' - 5y' + 6y = e^(4x).

Solusi homogen: yₕ = C₁e^(2x) + C₂e^(3x).

Tebak yₚ = Ae^(4x). yₚ' = 4Ae^(4x). yₚ'' = 16Ae^(4x).

Substitusi: 16Ae^(4x) - 20Ae^(4x) + 6Ae^(4x) = e^(4x)
2Ae^(4x) = e^(4x) → A = 1/2.

yₚ = (1/2)e^(4x).

Solusi umum: y = C₁e^(2x) + C₂e^(3x) + (1/2)e^(4x). ✅

4. Transformasi Laplace — Dari Kalkulus ke Aljabar

Transformasi Laplace mengubah fungsi waktu f(t) menjadi fungsi frekuensi kompleks F(s):

ℒ{f(t)} = F(s) = ∫₀^∞ f(t) e^(-st) dt

Keajaibannya: Transformasi Laplace mengubah persamaan diferensial menjadi persamaan aljabar! Jauh lebih mudah diselesaikan.

Mengapa ini bekerja? Transformasi Laplace "memakan" turunan dan mengeluarkan perkalian dengan s:

ℒ{f'(t)} = sF(s) - f(0)
ℒ{f''(t)} = s²F(s) - sf(0) - f'(0)
ℒ{f⁽ⁿ⁾(t)} = sⁿF(s) - sⁿ⁻¹f(0) - ... - f⁽ⁿ⁻¹⁾(0)

Bukti untuk turunan pertama:

ℒ{f'(t)} = ∫₀^∞ f'(t)e^(-st) dt.

Integral parsial: u = e^(-st), dv = f'(t)dt.
du = -se^(-st)dt, v = f(t).

= [f(t)e^(-st)]₀^∞ + s∫₀^∞ f(t)e^(-st) dt
= (0 - f(0)) + sF(s) = sF(s) - f(0). ✅
Soal: Selesaikan y'' + 4y = 0, y(0) = 1, y'(0) = 0 dengan Transformasi Laplace.

ℒ{y''} + 4ℒ{y} = 0
(s²Y - s·1 - 0) + 4Y = 0
Y(s² + 4) = s
Y = s/(s² + 4).

Transformasi invers: ℒ⁻¹{s/(s²+4)} = cos 2t. ✅

Solusi: y(t) = cos 2t. Bandingkan dengan metode karakteristik—hasilnya sama!

5. Deret Fourier — Menguraikan Sinyal Menjadi Gelombang

Deret Fourier menyatakan fungsi periodik f(x) dengan periode 2L sebagai jumlah tak hingga sinus dan cosinus:

f(x) = a₀/2 + Σ[n=1..∞] (aₙ cos(nπx/L) + bₙ sin(nπx/L))

aₙ = (1/L) ∫₋ᴸᴸ f(x) cos(nπx/L) dx
bₙ = (1/L) ∫₋ᴸᴸ f(x) sin(nπx/L) dx

Mengapa ini revolusioner? Joseph Fourier (1807) mengklaim bahwa fungsi periodik APAPUN—bahkan yang tidak mulus—bisa dinyatakan sebagai jumlah gelombang sinus dan cosinus. Ini mengubah cara kita memahami suara, cahaya, panas, dan sinyal.

Intuisi: Setiap bunyi musik adalah gabungan dari nada-nada murni (sinus dan cosinus). Deret Fourier adalah "resep" untuk menguraikan bunyi kompleks menjadi komponen-komponen frekuensinya. Inilah dasar dari MP3, JPEG, dan semua kompresi digital.

Soal: Cari deret Fourier untuk fungsi gelombang persegi f(x) dengan periode 2π:
f(x) = 1 untuk 0 < x < π, f(x) = -1 untuk -π < x < 0.

f ganjil → aₙ = 0 untuk semua n (termasuk a₀ = 0).

bₙ = (1/π)∫₋ᵖⁱᵖⁱ f(x) sin(nx) dx = (2/π)∫₀ᵖⁱ sin(nx) dx
= (2/π)[-cos(nx)/n]₀ᵖⁱ = (2/π)(-cos(nπ)/n + 1/n) = (2/(nπ))(1 - cos(nπ)).

cos(nπ) = (-1)ⁿ. Untuk n genap: 1 - 1 = 0. Untuk n ganjil: 1 - (-1) = 2.

bₙ = 4/(nπ) untuk n ganjil.

f(x) = (4/π)[sin x + (1/3)sin 3x + (1/5)sin 5x + (1/7)sin 7x + ...]. ✅

Gelombang persegi = jumlah gelombang sinus ganjil dengan amplitudo menurun!

6. Differential Forms — Pengantar Geometri Diferensial

Differential forms adalah generalisasi dari konsep dx, dy, ds, dA, dV ke dimensi berapapun. Mereka adalah "objek yang siap diintegralkan" pada manifold (ruang melengkung).

0-form: fungsi skalar f(x, y, z).
1-form: ω = P dx + Q dy + R dz (integral garis).
2-form: ω = P dy∧dz + Q dz∧dx + R dx∧dy (integral permukaan).
3-form: ω = f dx∧dy∧dz (integral volume).

∧ (wedge product): perkalian anti-komutatif: dx∧dy = -dy∧dx, dx∧dx = 0.

Mengapa ini penting? Differential forms menyatukan semua teorema integral (Green, Stokes, Gauss) menjadi satu teorema: Generalized Stokes' Theorem.

∫_M dω = ∫_{∂M} ω

d = operator turunan luar (exterior derivative).
M = manifold (ruang berdimensi k). ∂M = batas M (berdimensi k-1).

• FTC: M = [a,b] (1D), ω = f (0-form), dω = f'(x)dx, ∂M = {a,b}.
• Green: M = D (2D), ω = Pdx+Qdy (1-form), dω = (∂Q/∂x-∂P/∂y)dx∧dy.
• Stokes: M = S (permukaan 2D), ω = F⃗·dr⃗ (1-form), dω = curl F⃗·dS⃗.
• Gauss: M = E (benda 3D), ω = F⃗·dS⃗ (2-form), dω = div F⃗ dV.

Semua teorema integral adalah satu teorema yang sama!

📝 Ringkasan Cepat Kalkulus 11

  • PD Orde Satu Separable: Pisahkan variabel, integralkan. dy/dx = g(x)h(y).
  • PD Linear Orde Satu: Faktor integrasi μ = e^(∫Pdx). d(μy)/dx = μQ.
  • PD Orde Dua Homogen: Persamaan karakteristik ar²+br+c=0. 3 kasus (real, kembar, kompleks).
  • PD Orde Dua Non-Homogen: y = yₕ + yₚ. Tebak yₚ berdasarkan f(x). Koefisien tak tentu.
  • Transformasi Laplace: ℒ{f(t)} = ∫₀^∞ f(t)e^(-st)dt. Mengubah PD → aljabar. ℒ{f'} = sF - f(0).
  • Deret Fourier: f(x) = a₀/2 + Σ(aₙ cos nx + bₙ sin nx). Menguraikan sinyal periodik.
  • Differential Forms: Generalized Stokes' Theorem ∫_M dω = ∫_{∂M} ω. Menyatukan Green, Stokes, Gauss.

✨ Tips & Trik

  • PD Separable: kunci adalah memisahkan. Kalau kamu bisa menulis semua y di satu sisi dan semua x di sisi lain, kamu bisa menyelesaikannya. Ini adalah jenis PD paling dasar.
  • Faktor integrasi selalu e^(∫Pdx). Jangan lupa: P(x) adalah koefisien y setelah PD ditulis dalam bentuk standar dy/dx + Py = Q.
  • PD Orde Dua: cek diskriminan dulu. D = b² - 4ac menentukan jenis solusi: D>0 (real), D=0 (kembar), D<0 (kompleks = osilasi).
  • Laplace: turunan jadi perkalian dengan s. Ini adalah "trik sulap" terbesar. PD yang rumit berubah jadi persamaan aljabar yang sederhana.
  • Fourier: fungsi ganjil → hanya sinus. Fungsi genap → hanya cosinus. Ini menghemat setengah perhitungan! Cek simetri fungsi sebelum menghitung koefisien.
  • Differential forms: dx∧dy = -dy∧dx. Sifat anti-komutatif ini adalah kunci. Ini yang membuat dx∧dx = 0 dan menghasilkan determinan Jacobian.
  • Semua teorema integral = Generalized Stokes. Kalau kamu belajar differential forms, kamu tidak perlu menghafal Green, Stokes, Gauss secara terpisah. Satu rumus untuk semua!
