Part 7 • Aplikasi Integral: Luas, Volume, Panjang & Pusat Massa

📖 Pendahuluan — Integral Bukan Cuma Teori

Di Kalkulus 5 dan 6, kita sudah menguasai cara menghitung integral— dari substitusi dasar sampai teknik-teknik advance. Sekarang saatnya menuai hasil: menerapkan integral untuk memecahkan masalah nyata.

Aplikasi integral sangat luas. Insinyur menggunakannya untuk menghitung volume tangki, arsitek untuk menghitung luas permukaan kubah, fisikawan untuk mencari pusat massa dan momen inersia benda. Setiap kali kamu melihat benda melengkung—jembatan gantung, botol, bola, bahkan bentuk telur—integral adalah alat untuk mengukurnya.

Di bab ini, kita akan membahas enam aplikasi fundamental integral: luas daerah (termasuk daerah yang dibatasi dua kurva), volume benda putar (dengan metode cakram, cincin, dan kulit tabung), panjang kurva, luas permukaan benda putar, pusat massa (titik keseimbangan), dan momen inersia (ukuran resistansi terhadap rotasi).

Tujuan Bab
  • Mampu menghitung luas daerah yang dibatasi oleh kurva.
  • Menguasai perhitungan volume benda putar dengan metode cakram dan kulit tabung.
  • Memahami kapan menggunakan metode cakram vs metode kulit.
  • Mampu menghitung panjang kurva (arc length).
  • Mampu menghitung luas permukaan benda putar.
  • Memahami konsep pusat massa dan cara menghitungnya.
  • Mengenal momen inersia sebagai ukuran resistansi rotasi.

1. Luas Daerah di Bawah Kurva — Kembali ke Akar Integral

Luas daerah antara kurva y = f(x) dan sumbu X dari x = a ke x = b:

L = ∫ₐᵇ |f(x)| dx

Nilai mutlak diperlukan karena luas selalu positif. Jika f(x) bisa negatif di sebagian interval, kita harus memecah integral atau menggunakan |f(x)|.

1.1 Luas dengan Satu Kurva — Hati-hati dengan Bagian Negatif!

Hitung luas daerah antara y = x² - 4 dan sumbu X dari x = 0 ke x = 3.

f(0) = -4 (negatif), f(2) = 0, f(3) = 5 (positif).
Kurva memotong sumbu X di x = 2.

Pecah integral:
L = ∫₀² -(x²-4) dx + ∫₂³ (x²-4) dx
= ∫₀² (4-x²) dx + ∫₂³ (x²-4) dx

= [4x - x³/3]₀² + [x³/3 - 4x]₂³
= (8 - 8/3) + (27/3 - 12) - (8/3 - 8)
= (16/3) + (9 - 12) - (8/3 - 24/3)
= 16/3 + (-3) - (-16/3)
= 16/3 - 3 + 16/3 = 32/3 - 3 = 23/3 ≈ 7,67 ✅

Perhatikan: Jika langsung ∫₀³ (x²-4)dx tanpa nilai mutlak, hasilnya = [x³/3 - 4x]₀³ = (9-12) - 0 = -3. Ini SALAH karena luas tidak mungkin negatif!

1.2 Luas Antara Dua Kurva — Kurva Atas Dikurangi Kurva Bawah

Luas daerah antara dua kurva y = f(x) (atas) dan y = g(x) (bawah) dari x = a ke x = b:

L = ∫ₐᵇ [f(x) - g(x)] dx

f(x) ≥ g(x) pada [a, b].

Mengapa rumusnya pengurangan? Karena luas di bawah f(x) dikurangi luas di bawah g(x) menyisakan luas di antara keduanya. Ini seperti menghitung luas "selaput" di antara dua kurva.

Hitung luas daerah antara y = x² dan y = x + 2 dari titik potong ke titik potong.

Cari titik potong: x² = x + 2 → x² - x - 2 = 0 → (x-2)(x+1) = 0.
Titik potong di x = -1 dan x = 2.

Untuk x ∈ [-1, 2]: mana yang di atas? Uji x = 0: f(0) = 0, g(0) = 2 → g di atas!
Jadi f(x) = x+2 (atas), g(x) = x² (bawah).

L = ∫₋₁² [(x+2) - x²] dx = ∫₋₁² (x + 2 - x²) dx
= [x²/2 + 2x - x³/3]₋₁²
= (4/2 + 4 - 8/3) - (1/2 - 2 + 1/3)
= (2 + 4 - 8/3) - (1/2 - 2 + 1/3)
= (6 - 8/3) - (-3/2 + 1/3)
= (18/3 - 8/3) - (-9/6 + 2/6)
= 10/3 - (-7/6) = 10/3 + 7/6 = 20/6 + 7/6 = 27/6 = 9/2 = 4,5 ✅

1.3 Luas Terhadap Sumbu Y — Ketika x Dinyatakan dalam y

Kadang lebih mudah mengintegralkan terhadap y daripada x.

L = ∫ₐᵇ [f(y) - g(y)] dy

di mana f(y) adalah kurva kanan, g(y) adalah kurva kiri.
Hitung luas daerah antara x = y² dan x = y + 2.

Titik potong: y² = y + 2 → y² - y - 2 = 0 → (y-2)(y+1) = 0 → y = -1, 2.

Untuk y ∈ [-1, 2]: mana yang di kanan? Uji y = 0: x_kiri = 0, x_kanan = 2 → x = y+2 di kanan.

L = ∫₋₁² [(y+2) - y²] dy = [y²/2 + 2y - y³/3]₋₁²
= (2 + 4 - 8/3) - (1/2 - 2 + 1/3) = 9/2 = 4,5 ✅

Hasilnya sama dengan contoh sebelumnya—hanya variabel integrasinya yang berbeda!

2. Volume Benda Putar — Memutar Kurva Menjadi Benda 3D

Bayangkan sebuah kurva diputar 360° mengelilingi suatu sumbu. Hasilnya adalah benda putar (solid of revolution)—benda tiga dimensi yang simetris terhadap sumbu putar. Contoh: bola (dari setengah lingkaran), kerucut (dari garis miring), tabung (dari garis horizontal).

2.1 Metode Cakram (Disk Method) — Irisan Berbentuk Lingkaran

Metode Cakram digunakan ketika sumbu putar sejajar dengan arah irisan.

Putar terhadap sumbu X (dari y = f(x)):
V = ∫ₐᵇ π[f(x)]² dx   (satu kurva, dari x=a ke x=b)

Putar terhadap sumbu Y (dari x = g(y)):
V = ∫ₐᵇ π[g(y)]² dy

Mengapa rumusnya π[f(x)]²? Setiap irisan tegak lurus sumbu putar berbentuk lingkaran dengan jari-jari f(x). Luas lingkaran = πr² = π[f(x)]². Volume = jumlah luas irisan × ketebalan = ∫ π[f(x)]² dx.

Soal: Kurva y = √x dari x = 0 ke x = 4 diputar mengelilingi sumbu X. Hitung volume benda putarnya.

V = ∫₀⁴ π(√x)² dx = π ∫₀⁴ x dx = π [x²/2]₀⁴ = π(16/2) = 8π ✅

Bentuknya seperti mangkuk parabola (paraboloid).

2.2 Metode Cincin (Washer Method) — Untuk Daerah Antara Dua Kurva

Jika daerah antara dua kurva diputar, irisannya berbentuk cincin (lingkaran besar dikurangi lingkaran kecil).

V = ∫ₐᵇ π[(f(x))² - (g(x))²] dx

f(x) = jari-jari luar, g(x) = jari-jari dalam.
Soal: Daerah antara y = x² dan y = x dari x = 0 ke x = 1 diputar mengelilingi sumbu X.

Untuk x ∈ [0,1]: x ≥ x², jadi f(x) = x (jari-jari luar), g(x) = x² (jari-jari dalam).

V = ∫₀¹ π[x² - (x²)²] dx = π ∫₀¹ (x² - x⁴) dx
= π [x³/3 - x⁵/5]₀¹ = π(1/3 - 1/5) = π(5/15 - 3/15) = 2π/15 ✅

2.3 Metode Kulit Tabung (Shell Method) — Alternatif yang Sering Lebih Mudah

Metode Kulit Tabung digunakan ketika sumbu putar tegak lurus arah irisan, atau ketika fungsi sulit dinyatakan dalam variabel yang diinginkan.

Putar terhadap sumbu Y (dari y = f(x)):
V = ∫ₐᵇ 2πx·f(x) dx

Intuisi: Setiap "kulit" adalah selimut tabung dengan jari-jari x, tinggi f(x), dan ketebalan dx. Luas selimut = 2πx × f(x). Volume = luas × ketebalan.

Mengapa 2πx·f(x)? Bayangkan kamu membuka kulit tabung dan meratakannya. Kamu dapat persegi panjang dengan lebar = keliling lingkaran (2πx) dan tinggi = f(x). Luasnya 2πx·f(x). Dikalikan ketebalan dx, itulah volume kulit.

Soal: Daerah di bawah y = x² dari x = 0 ke x = 2 diputar mengelilingi sumbu Y. Hitung volumenya.

Metode Kulit:
V = ∫₀² 2πx·x² dx = 2π ∫₀² x³ dx = 2π [x⁴/4]₀² = 2π(16/4) = 8π ✅

Bandingkan dengan metode cakram terhadap Y:
y = x² → x = √y. Batas y: 0 ke 4.
V = ∫₀⁴ π(√y)² dy = π ∫₀⁴ y dy = π[y²/2]₀⁴ = π(16/2) = 8π. Sama!

Tapi metode kulit jauh lebih langsung karena tidak perlu mengubah fungsi ke x = g(y).

2.4 Kapan Pakai Cakram vs Kulit?

| Situasi | Metode | |---|---| | Sumbu putar sejajar irisan & fungsi mudah dikuadratkan | Cakram/Cincin | | Sumbu putar tegak lurus irisan | Kulit Tabung | | Fungsi sulit diubah ke variabel lain (misal y = x³+x) | Kulit Tabung | | Daerah antara dua kurva dengan sumbu putar di antaranya | Cincin |

3. Panjang Kurva (Arc Length) — Mengukur Garis Lengkung

Panjang kurva y = f(x) dari x = a ke x = b:

L = ∫ₐᵇ √(1 + [f'(x)]²) dx

Intuisi: Potong kurva menjadi segmen-segmen sangat kecil. Setiap segmen hampir lurus. Panjangnya ≈ √(dx² + dy²) = √(1 + (dy/dx)²) dx. Jumlahkan semua segmen → integral.

Mengapa ada √(1 + [f'(x)]²)? — Teorema Pythagoras pada Segmen Kecil

Bayangkan satu segmen kecil pada kurva. Perubahan horizontal = dx. Perubahan vertikal = dy = f'(x)dx.

Dengan Pythagoras: panjang segmen = √(dx² + dy²) = √(dx² + (f'(x)dx)²)
= √(1 + [f'(x)]²) · dx.

Integralkan dari a ke b → total panjang kurva. ✅
Hitung panjang kurva y = x^(3/2) dari x = 0 ke x = 4.

f'(x) = (3/2)x^(1/2). [f'(x)]² = (9/4)x.

L = ∫₀⁴ √(1 + 9x/4) dx.

Substitusi u = 1 + 9x/4 → du = (9/4)dx → dx = (4/9)du.
Saat x = 0: u = 1. Saat x = 4: u = 1 + 9 = 10.

L = ∫₁¹⁰ √u · (4/9) du = (4/9) ∫₁¹⁰ u^(1/2) du
= (4/9) [u^(3/2)/(3/2)]₁¹⁰ = (4/9) · (2/3) [u^(3/2)]₁¹⁰
= (8/27)(10^(3/2) - 1) ≈ (8/27)(31,623 - 1) ≈ (8/27)(30,623) ≈ 9,07 ✅

4. Luas Permukaan Benda Putar — "Kulit" Benda 3D

Luas permukaan yang dihasilkan ketika kurva y = f(x) diputar mengelilingi sumbu X dari x = a ke x = b:

S = ∫ₐᵇ 2π f(x) √(1 + [f'(x)]²) dx

Intuisi: Setiap "pita" kecil pada kurva, ketika diputar, membentuk selimut kerucut terpancung dengan jari-jari f(x) dan panjang sisi miring √(1 + [f'(x)]²) dx. Luas pita = 2πr × panjang miring.
Hitung luas permukaan bola berjari-jari R (dari setengah lingkaran y = √(R²-x²)).

f(x) = √(R²-x²), domain x ∈ [-R, R].
f'(x) = -x/√(R²-x²).

1 + [f'(x)]² = 1 + x²/(R²-x²) = (R²-x²+x²)/(R²-x²) = R²/(R²-x²).
√(1+[f'(x)]²) = R/√(R²-x²).

S = ∫₋ᴿᴿ 2π √(R²-x²) · R/√(R²-x²) dx = ∫₋ᴿᴿ 2πR dx
= 2πR [x]₋ᴿᴿ = 2πR(R - (-R)) = 4πR² ✅

Ini adalah rumus luas permukaan bola yang legendaris! Diturunkan dengan elegan menggunakan integral.

5. Pusat Massa (Centroid) — Titik Keseimbangan

Pusat massa (centroid) adalah titik di mana seluruh massa benda "terkonsentrasi" untuk tujuan perhitungan keseimbangan. Jika benda ditopang tepat di pusat massanya, ia akan seimbang sempurna.

Untuk daerah datar (lamina) dengan densitas seragam yang dibatasi oleh y = f(x) dan y = g(x) dari x = a ke x = b:

Massa: M = ρ ∫ₐᵇ [f(x) - g(x)] dx   (ρ = densitas, biasanya ρ=1 untuk centroid)

Momen terhadap sumbu Y: Mᵧ = ρ ∫ₐᵇ x[f(x) - g(x)] dx

Momen terhadap sumbu X: Mₓ = ρ ∫ₐᵇ (1/2)[f(x)² - g(x)²] dx

Koordinat pusat massa: x̄ = Mᵧ/M,   ȳ = Mₓ/M

Mengapa momen terhadap X memakai (1/2)[f(x)² - g(x)²]?

Momen adalah "berat × jarak". Untuk setiap irisan vertikal tipis, jarak rata-rata titik-titik di irisan itu ke sumbu X adalah rata-rata f(x) dan g(x) = (f(x)+g(x))/2.

Maka momen irisan = (f(x)+g(x))/2 × (f(x)-g(x))dx × ρ
= (ρ/2)(f(x)² - g(x)²)dx. ✅
Cari pusat massa daerah antara y = x² dan y = x dari x = 0 ke x = 1 (ρ = 1).

Massa: M = ∫₀¹ (x - x²) dx = [x²/2 - x³/3]₀¹ = 1/2 - 1/3 = 1/6.

Mᵧ: ∫₀¹ x(x - x²) dx = ∫₀¹ (x² - x³) dx = [x³/3 - x⁴/4]₀¹ = 1/3 - 1/4 = 1/12.

Mₓ: (1/2)∫₀¹ (x² - x⁴) dx = (1/2)[x³/3 - x⁵/5]₀¹ = (1/2)(1/3 - 1/5) = (1/2)(2/15) = 1/15.

x̄ = (1/12)/(1/6) = 1/2. ȳ = (1/15)/(1/6) = 6/15 = 2/5.

Pusat massa: (1/2, 2/5). ✅

6. Momen Inersia — Resistansi Terhadap Rotasi

Momen inersia (I) mengukur seberapa sulit memutar suatu benda terhadap sumbu tertentu. Mirip dengan massa untuk gerak translasi, momen inersia adalah "massa rotasional".

Untuk daerah datar dengan densitas ρ = 1:

Terhadap sumbu X: Iₓ = ∫ₐᵇ (1/3)[f(x)³ - g(x)³] dx

Terhadap sumbu Y: Iᵧ = ∫ₐᵇ x²[f(x) - g(x)] dx

Momen inersia polar (terhadap titik asal): I₀ = Iₓ + Iᵧ

Mengapa Iₓ memakai (1/3)[f(x)³ - g(x)³]?

Momen inersia adalah integral dari (jarak)² × massa.
Untuk irisan vertikal, jarak titik ke sumbu X adalah y. Rata-rata y² di irisan = ∫ y² dy / Δy = (f³-g³)/(3(f-g)).
Kalikan dengan massa irisan (f-g)dx → (f³-g³)/3 dx. ✅
Hitung momen inersia terhadap sumbu Y untuk daerah antara y = x² dan y = x dari 0 ke 1.

Iᵧ = ∫₀¹ x²(x - x²) dx = ∫₀¹ (x³ - x⁴) dx
= [x⁴/4 - x⁵/5]₀¹ = 1/4 - 1/5 = 5/20 - 4/20 = 1/20 ✅

📝 Ringkasan Cepat Kalkulus 7

  • Luas daerah: L = ∫ₐᵇ [f(x)-g(x)] dx. Bisa terhadap x atau y. Hati-hati dengan bagian negatif!
  • Volume — Cakram: V = ∫ π[f(x)]² dx. Sumbu putar sejajar irisan.
  • Volume — Cincin: V = ∫ π[f(x)² - g(x)²] dx. Daerah antara dua kurva.
  • Volume — Kulit Tabung: V = ∫ 2πx·f(x) dx. Sumbu putar tegak lurus irisan.
  • Panjang kurva: L = ∫ √(1 + [f'(x)]²) dx. Pythagoras pada segmen kecil.
  • Luas permukaan putar: S = ∫ 2πf(x)√(1+[f'(x)]²) dx.
  • Pusat massa: x̄ = Mᵧ/M, ȳ = Mₓ/M. Titik keseimbangan benda.
  • Momen inersia: I = ∫ (jarak)² dm. Resistansi terhadap rotasi.

✨ Tips & Trik

  • Luas: gambar dulu! Sebelum menghitung, sketsa kurvanya. Tentukan mana yang di atas, mana yang di bawah, dan di mana titik potongnya.
  • Luas: periksa tanda! Jika kurva di bawah sumbu X, integralnya negatif. Luas selalu positif → pakai nilai mutlak atau pecah integral.
  • Volume: pilih metode dengan bijak. Cakram untuk sumbu sejajar irisan. Kulit untuk sumbu tegak lurus irisan atau fungsi sulit diinverskan.
  • Jangan lupa π di volume! Luas lingkaran = πr². Kesalahan paling umum adalah lupa mengalikan dengan π.
  • Panjang kurva: turunan dulu, baru akar. Hitung f'(x), kuadratkan, tambah 1, akarkan, baru integralkan. Urutannya penting.
  • Pusat massa: momen dibagi massa. Mᵧ/M untuk x̄, Mₓ/M untuk ȳ. Jangan tertukar! Mᵧ = ∫ x·(f-g)dx, Mₓ = (1/2)∫ (f²-g²)dx.
  • Verifikasi dengan simetri. Jika daerah simetris terhadap sumbu Y, maka x̄ = 0. Jika simetris terhadap garis y = c, maka ȳ = c.
