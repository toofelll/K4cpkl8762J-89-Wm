Distribusi Kontinu & Teorema Limit

Pengantar Distribusi Kontinu

Distribusi kontinu berbeda dari diskrit karena variabel acaknya tidak lagi berupa hitungan, tetapi nilai dalam suatu interval. Artinya, peluang tidak dihitung untuk satu titik, tetapi untuk suatu rentang.

Intuisi Kunci

Untuk variabel kontinu: P(X = x) = 0 yang bermakna kita selalu menghitung peluang dalam interval: P(a ≤ X ≤ b)

7.1 Distribusi Uniform Kontinu

Distribusi Uniform Kontinu adalah distribusi paling sederhana dalam dunia kontinu. Semua nilai dalam interval tertentu memiliki peluang yang sama “kepadatan”-nya.

Intuisi Kunci

Jika sebuah variabel acak X tersebar merata di interval [a, b], maka tidak ada nilai yang lebih mungkin dari yang lain. Yang penting bukan titik, tetapi panjang interval.

Definisi

X ~ U(a, b)

dengan interval:

a ≤ X ≤ b

Fungsi PDF

Probabilitas density function (PDF):

f(x) = 1 / (b - a), untuk a ≤ x ≤ b

dan 0 di luar interval.

Peluang Interval

Peluang dihitung sebagai luas:

P(c ≤ X ≤ d) = (d - c) / (b - a)

Contoh 1 — Waktu Kedatangan

Seseorang datang secara acak antara jam 1 sampai 2 siang. Tentukan peluang datang antara menit ke-10 sampai ke-30.

a = 0, b = 60

P = (30 - 10) / 60 = 20/60 = 1/3

Jawaban: 1/3

Nilai Harapan

E(X) = (a + b) / 2

Artinya titik tengah interval.

Variansi

Var(X) = (b - a)² / 12

Kesalahan Umum

Banyak siswa mencoba menghitung peluang titik. Padahal dalam kontinu: peluang titik selalu 0.

7.2 Distribusi Eksponensial

Distribusi Eksponensial digunakan untuk memodelkan waktu tunggu sampai suatu kejadian terjadi. Ini adalah versi kontinu dari ide “menunggu sukses pertama”.

Intuisi Kunci

Jika kejadian terjadi secara acak dalam waktu, maka kita tidak menghitung jumlah kejadian, tetapi: berapa lama sampai kejadian berikutnya terjadi.

Parameter

λ (lambda) = laju kejadian per satuan waktu

Definisi

X ~ Exp(λ)

X = waktu tunggu sampai kejadian pertama.

Fungsi PDF

f(x) = λ e^{-λx}, x ≥ 0

dan f(x) = 0 untuk x < 0.

Fungsi Peluang (CDF)

P(X ≤ x) = 1 - e^{-λx}

Contoh 1 — Kedatangan Bus

Bus datang rata-rata 3 kali per jam. Tentukan peluang menunggu kurang dari 10 menit.

λ = 3 per jam x = 10 menit = 1/6 jam

P(X ≤ x) = 1 - e^{-3(1/6)}

= 1 - e^{-1/2}

Contoh 2 — Error Sistem

Rata-rata error terjadi 2 per jam. Peluang waktu sampai error pertama lebih dari 1 jam:

P(X > 1) = e^{-2}

Sifat Penting (Memoryless)

Sifat Unik

Distribusi eksponensial memiliki sifat:

P(X > s + t | X > s) = P(X > t)

Artinya: masa lalu tidak mempengaruhi masa depan.

Nilai Harapan

E(X) = 1 / λ

Variansi

Var(X) = 1 / λ²

Kesalahan Umum

Banyak siswa bingung antara Poisson dan Eksponensial:

  • Poisson → jumlah kejadian
  • Eksponensial → waktu antar kejadian

7.3 Distribusi Normal

Distribusi Normal adalah distribusi paling penting dalam statistika dan probabilitas. Banyak fenomena alami—tinggi badan, nilai ujian, error pengukuran—cenderung mengikuti distribusi ini.

Intuisi Kunci

Distribusi Normal berbentuk kurva lonceng (bell curve), simetris terhadap mean. Sebagian besar data terkumpul di sekitar rata-rata, dan semakin jauh dari rata-rata, semakin kecil peluangnya.

Notasi

X ~ N(μ, σ²)

di mana:

  • μ = mean (rata-rata)
  • σ = standar deviasi

Fungsi PDF

f(x) = (1 / (σ√(2π))) · e^{-(x - μ)² / (2σ²)}

Sifat Penting

  • Simetris terhadap μ
  • Mean = Median = Modus
  • Total luas kurva = 1

Standarisasi (Z-Score)

Untuk menghitung peluang, kita ubah ke standar normal:

Z = (X - μ) / σ

Makna Z-Score

Z menunjukkan seberapa jauh nilai X dari rata-rata dalam satuan standar deviasi.

Contoh 1 — Nilai Ujian

Nilai ujian berdistribusi normal dengan:

μ = 70, σ = 10

Tentukan Z untuk nilai 85:

Z = (85 - 70) / 10 = 1.5

Contoh 2 — Interpretasi

Jika Z = 0 → tepat di rata-rata. Jika Z > 0 → di atas rata-rata. Jika Z < 0 → di bawah rata-rata.

Kesalahan Umum

Banyak siswa mengira semua distribusi simetris adalah normal. Padahal normal memiliki struktur matematis khusus (eksponensial kuadrat).

7.4 Central Limit Theorem (CLT)

Central Limit Theorem adalah salah satu teorema paling penting dalam probabilitas. Intinya: jumlah dari banyak variabel acak akan mendekati distribusi normal, meskipun distribusi asalnya tidak normal.

Intuisi Kunci

Jika kita menjumlahkan banyak “kejadian acak kecil”, maka hasil akhirnya akan membentuk pola lonceng (normal). Ini berlaku sangat luas dalam matematika, fisika, ekonomi, dan statistik.

Ide Dasar

Misalkan kita punya variabel acak independen:

X₁, X₂, ..., Xₙ

dengan mean μ dan variansi σ².

Jika kita definisikan jumlahnya:

Sₙ = X₁ + X₂ + ... + Xₙ

Bentuk CLT

Ketika n besar:

Sₙ ~ N(nμ, nσ²)

atau dalam bentuk standar:

Z = (Sₙ - nμ) / (σ√n)

Makna Penting

CLT menjelaskan mengapa distribusi normal begitu sering muncul di dunia nyata. Karena banyak proses adalah hasil penjumlahan dari banyak faktor kecil.

Contoh Intuisi

Misalkan kita melempar dadu 100 kali. Masing-masing hasil acak, tetapi jika dijumlahkan:

  • hasil total akan mendekati distribusi normal
  • meskipun dadu itu sendiri tidak normal

Aplikasi Olimpiade

CLT dipakai untuk:

  • aproksimasi Binomial besar
  • analisis rata-rata sampel
  • estimasi probabilitas kompleks

Kesalahan Umum

  • Mengira CLT hanya berlaku untuk normal awal
  • Mengabaikan syarat independensi
  • Menggunakan CLT untuk n kecil

7.5 Hukum Bilangan Besar (Law of Large Numbers)

Hukum Bilangan Besar menjelaskan bagaimana rata-rata hasil percobaan acak akan mendekati nilai harapan ketika jumlah percobaan semakin besar. Ini adalah fondasi penting yang menjelaskan mengapa probabilitas “stabil” dalam jangka panjang.

Intuisi Kunci

Jika sebuah percobaan diulang berkali-kali, maka: rata-rata hasil observasi akan mendekati nilai ekspektasi (E(X)).

Bentuk Matematis

Misalkan X₁, X₂, ..., Xₙ adalah variabel acak independen identik dengan mean μ. Definisikan rata-rata sampel:

X̄ₙ = (X₁ + X₂ + ... + Xₙ) / n

Maka:

X̄ₙ → μ (ketika n → ∞)

Makna Penting

LLN menjelaskan bahwa: meskipun hasil individual acak, rata-ratanya menjadi stabil dalam jangka panjang.

Contoh Intuisi

Jika kita melempar koin:

  • 10 kali → bisa sangat tidak stabil
  • 100 kali → mulai stabil
  • 10.000 kali → sangat dekat 50% kepala

LLN vs CLT

Hukum Bilangan Besar Central Limit Theorem
Rata-rata → mendekati mean Distribusi jumlah → mendekati normal
Fokus pada konvergensi Fokus pada bentuk distribusi
Menjawab “menuju ke mana” Menjawab “bentuknya seperti apa”

Aplikasi Olimpiade

LLN sering digunakan untuk:

  • estimasi probabilitas dari eksperimen besar
  • validasi model peluang
  • intuisi kestabilan statistik

Kesalahan Umum

  • Mengira LLN berlaku pada jumlah kecil
  • Menyamakan LLN dengan CLT
  • Mengabaikan bahwa konvergensi terjadi saat n besar

7.x Peta Besar Distribusi Kontinu & Teorema Limit

Setelah mempelajari seluruh distribusi kontinu dan teorema limit, bagian ini menyatukan semua konsep menjadi satu struktur besar. Tujuannya agar kamu tidak hanya tahu rumus, tetapi tahu hubungan antar konsep.

Struktur Besar

Distribusi kontinu dan teorema limit dapat dipahami sebagai tiga lapisan besar:

  • Lapisan 1 → Distribusi dasar (Uniform, Eksponensial, Normal)
  • Lapisan 2 → Struktur proses acak (waktu & skala)
  • Lapisan 3 → Perilaku jangka panjang (CLT & LLN)

1. Distribusi Uniform Kontinu

Semua nilai dalam interval memiliki peluang yang sama. Ini adalah bentuk paling sederhana dari distribusi kontinu.

U(a, b)

2. Distribusi Eksponensial

Model waktu tunggu antar kejadian acak.

f(x) = λe^{-λx}

Eksponensial adalah “pasangan waktu” dari Poisson (jumlah kejadian).

3. Distribusi Normal

Distribusi paling penting dalam statistika. Muncul dari penjumlahan banyak faktor kecil acak.

N(μ, σ²)

4. Central Limit Theorem (CLT)

CLT menjelaskan bahwa:

Jumlah banyak variabel acak independen akan mendekati distribusi normal, meskipun distribusi asalnya tidak normal.

5. Hukum Bilangan Besar (LLN)

LLN menjelaskan bahwa:

Rata-rata sampel akan mendekati nilai harapan (mean) ketika jumlah percobaan sangat besar.

Hubungan Besar Antar Konsep

Konsep Fokus Hasil
Uniform Kepastian peluang sama Distribusi dasar
Eksponensial Waktu tunggu Proses kejadian
Normal Bentuk distribusi Kurva lonceng
CLT Penjumlahan variabel Menuju normal
LLN Rata-rata sampel Menuju mean

Kesalahan Konseptual Umum

  • Mengira CLT dan LLN adalah hal yang sama
  • Menganggap normal hanya “kurva simetris”
  • Lupa bahwa eksponensial berhubungan dengan waktu, bukan jumlah

7.x Cheat Sheet Distribusi Kontinu & Limit

Bagian ini merangkum semua rumus penting untuk kecepatan olimpiade. Fokusnya bukan derivasi, tetapi hafalan pola + intuisi cepat.

1. Uniform Kontinu

X ~ U(a,b)

f(x) = 1/(b-a) P(c ≤ X ≤ d) = (d-c)/(b-a) E(X) = (a+b)/2 Var(X) = (b-a)²/12

2. Eksponensial

X ~ Exp(λ)

f(x) = λe^{-λx} P(X ≤ x) = 1 - e^{-λx} P(X > x) = e^{-λx} E(X) = 1/λ Var(X) = 1/λ²

Ingat: eksponensial = waktu tunggu

3. Normal

X ~ N(μ, σ²)

Z = (X - μ)/σ

Aturan penting:

  • ±1σ ≈ 68%
  • ±2σ ≈ 95%
  • ±3σ ≈ 99.7%

4. Central Limit Theorem (CLT)

Sₙ ≈ N(nμ, nσ²) Z = (Sₙ - nμ)/(σ√n)

Jumlah banyak variabel acak → normal

5. Hukum Bilangan Besar (LLN)

X̄ₙ → μ

Artinya: rata-rata sampel mendekati mean saat n besar.

Intuisi Super Cepat

  • Uniform → “semua sama mungkin”
  • Eksponensial → “berapa lama nunggu”
  • Normal → “bentuk alami data”
  • CLT → “penjumlahan jadi normal”
  • LLN → “rata-rata jadi stabil”

Kesalahan Cepat

  • Salah pakai Z-score
  • Lupa eksponensial adalah waktu
  • Menyamakan CLT dan LLN

7.x Strategi Pemilihan Distribusi (Olimpiade)

Bagian ini adalah inti “kecerdasan soal”. Bukan rumus yang menentukan jawaban, tetapi model yang kamu pilih. Salah memilih distribusi = seluruh jawaban salah.

Langkah Super Penting

  1. Baca proses acaknya
  2. Tentukan apa yang dihitung
  3. Cek apakah diskrit atau kontinu
  4. Cari keyword penting

Peta Keputusan Cepat

Kata Kunci / Situasi Distribusi
“satu percobaan” Bernoulli
“n percobaan, hitung sukses” Binomial
“sampai sukses pertama” Geometrik
“tanpa pengembalian” Hipergeometrik
“jumlah kejadian dalam waktu/ruang” Poisson
“semua sama peluang” Uniform
“waktu tunggu” Eksponensial
“penjumlahan banyak variabel” CLT → Normal
“rata-rata sampel besar” LLN

Jebakan Olimpiade

1. Binomial vs Hipergeometrik

Jika tidak ada pengembalian → bukan Binomial.

2. Poisson vs Eksponensial

  • Poisson → jumlah kejadian
  • Eksponensial → waktu antar kejadian

3. CLT vs LLN

  • CLT → bentuk distribusi
  • LLN → konvergensi rata-rata

Strategi Emas

Dalam soal sulit, jangan langsung hitung. Tanyakan dulu: “model acak apa yang sedang terjadi?”

Kesimpulan Penting

Distribusi bukan sekadar rumus. Distribusi adalah cara berpikir untuk memodelkan ketidakpastian.

Penutup Bab 7

Distribusi kontinu dan teorema limit adalah fondasi utama statistika modern. Dari Uniform hingga Normal, dari CLT hingga LLN, semuanya membentuk satu kesatuan besar: bagaimana dunia acak menjadi stabil ketika skalanya besar.

Ringkasan Akhir

  • Uniform → semua nilai sama peluang
  • Eksponensial → waktu tunggu kejadian
  • Normal → bentuk alami data
  • CLT → penjumlahan → normal
  • LLN → rata-rata → mean

Pesan Kunci

Probabilitas bukan hanya tentang menghitung peluang, tetapi tentang memahami struktur ketidakpastian.

Penutup Mindset

Jika kamu bisa mengenali model yang tepat, maka 80% soal sudah selesai bahkan sebelum menghitung.
