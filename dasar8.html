1. Nilai Mutlak — Jarak dari Nol

Nilai mutlak dari suatu bilangan adalah jarak bilangan itu dari nol pada garis bilangan. Jarak selalu bernilai positif atau nol.

|x| = x, jika x ≥ 0
|x| = -x, jika x < 0

Intuisi: Nilai mutlak "menghilangkan" tanda negatif. |5| = 5, |-5| = 5. Keduanya berjarak 5 satuan dari nol.

|7| = 7
|-7| = 7
|0| = 0
|-3,5| = 3,5
|2 - 5| = |-3| = 3 (jarak antara 2 dan 5 adalah 3)

Interpretasi geometris: |a - b| adalah jarak antara a dan b pada garis bilangan.

|8 - 3| = 5 → jarak antara 8 dan 3 adalah 5.
|3 - 8| = |-5| = 5 → sama, jarak tidak bergantung urutan.
|x - 4| = 3 → x adalah bilangan yang berjarak 3 dari 4, yaitu x = 1 atau x = 7.

Sifat-Sifat Nilai Mutlak

1. |x| ≥ 0 (selalu non-negatif)
2. |x| = 0 jika dan hanya jika x = 0
3. |x × y| = |x| × |y|
4. |x / y| = |x| / |y| (y ≠ 0)
5. |x + y| ≤ |x| + |y| (ketaksamaan segitiga)

Mengapa |x + y| ≤ |x| + |y|?

Bayangkan x = 5 dan y = -2. Maka: |x + y| = |5 - 2| = |3| = 3. |x| + |y| = 5 + 2 = 7. Memang 3 ≤ 7. Ketaksamaan ini disebut ketaksamaan segitiga karena analog dengan: panjang satu sisi segitiga selalu kurang dari atau sama dengan jumlah panjang dua sisi lainnya.

Persamaan dan Pertidaksamaan Nilai Mutlak

Persamaan:
|x| = a (a ≥ 0) → x = a atau x = -a

Pertidaksamaan:
|x| < a → -a < x < a (x di antara -a dan a)
|x| > a → x < -a atau x > a (x di luar interval [-a, a])
|x - 2| = 5 → x - 2 = 5 atau x - 2 = -5 → x = 7 atau x = -3

|x + 1| < 4 → -4 < x + 1 < 4 → -5 < x < 3

2. Interval — Menyatakan Rentang Bilangan

Interval adalah cara menyatakan suatu rentang bilangan pada garis bilangan.

Jenis-Jenis Interval

Interval tertutup [a, b]: semua x di mana a ≤ x ≤ b (termasuk ujung).
Interval terbuka (a, b): semua x di mana a < x < b (tidak termasuk ujung).
Interval setengah terbuka: [a, b) atau (a, b].
Interval tak hingga: [a, ∞), (-∞, b], (-∞, ∞) = semua bilangan real.

Simbol:

  • [ atau ] → termasuk batas (tertutup).
  • ( atau ) → tidak termasuk batas (terbuka).
  • ∞ → tak hingga (selalu pakai kurung biasa, karena tak hingga bukan bilangan).
[2, 5] = {x | 2 ≤ x ≤ 5} = {2, 2,1, 3, 4, 5, ...} termasuk 2 dan 5.

(2, 5) = {x | 2 < x < 5} = {2,1, 3, 4, 4,9, ...} tidak termasuk 2 dan 5.

[0, ∞) = {x | x ≥ 0} = semua bilangan non-negatif.

(-∞, 3) = {x | x < 3} = semua bilangan kurang dari 3.

Operasi pada Interval: Irisan dan Gabungan

Irisan: [1, 4] ∩ [3, 6] = [3, 4] (bagian yang tumpang tindih).

Gabungan: [1, 3] ∪ [5, 7] = {x | 1 ≤ x ≤ 3 atau 5 ≤ x ≤ 7}.

3. Himpunan — Koleksi Objek yang Terdefinisi Jelas

Himpunan adalah kumpulan objek-objek yang didefinisikan dengan jelas. Objek dalam himpunan disebut anggota atau elemen.

Cara Menyatakan Himpunan

1. Cara enumerasi (mendaftar):

A = {1, 2, 3, 4, 5}
B = {merah, kuning, hijau}
C = {2, 4, 6, 8, ...} (dengan pola)

2. Cara notasi pembentuk himpunan:

A = {x | x adalah bilangan bulat positif kurang dari 6}
B = {x | x > 0 dan x < 10}

Simbol "|" dibaca "di mana" atau "sedemikian sehingga".

Keanggotaan Himpunan

∈ = anggota (elemen)
∉ = bukan anggota
Misal A = {1, 2, 3}

2 ∈ A (2 adalah anggota A).
5 ∉ A (5 bukan anggota A).

Operasi Dasar Himpunan

Irisan (∩): A ∩ B = {x | x ∈ A dan x ∈ B}.
Gabungan (∪): A ∪ B = {x | x ∈ A atau x ∈ B}.
Selisih (-): A - B = {x | x ∈ A dan x ∉ B}.
Komplemen (Aᶜ): semua elemen yang bukan anggota A (dalam semesta tertentu).
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

A ∩ B = {3, 4}
A ∪ B = {1, 2, 3, 4, 5, 6}
A - B = {1, 2}
B - A = {5, 6}

Himpunan Bagian (Subset)

A ⊆ B artinya setiap anggota A juga anggota B.
A ⊂ B artinya A ⊆ B tapi A ≠ B (subset sejati).
{1, 2} ⊆ {1, 2, 3} → benar, semua anggota {1,2} ada di {1,2,3}.
{1, 2, 3} ⊆ {1, 2, 3} → benar, setiap himpunan subset dari dirinya sendiri.
{1, 4} ⊆ {1, 2, 3} → salah, 4 tidak ada di {1,2,3}.

4. Logika Matematika — Berpikir Benar-Salah

Logika matematika adalah fondasi dari semua penalaran matematika. Ia mempelajari aturan-aturan berpikir yang sah (valid).

Pernyataan (Proposisi)

Pernyataan adalah kalimat yang memiliki nilai kebenaran: benar (B) atau salah (S), tidak keduanya.

"2 + 2 = 4" → pernyataan, nilainya benar.
"7 adalah bilangan genap" → pernyataan, nilainya salah.
"x + 3 = 7" → bukan pernyataan (nilainya tergantung x).
"Halo, apa kabar?" → bukan pernyataan (kalimat tanya).

Negasi (Ingkaran)

Jika p adalah suatu pernyataan, maka negasi p (ditulis ¬p atau ~p) bernilai kebalikan dari p.
p: "Hari ini hujan."
¬p: "Hari ini tidak hujan."

p: "Semua bilangan prima ganjil." (salah, karena 2 prima tapi genap)
¬p: "Ada bilangan prima yang tidak ganjil." (benar)

Konjungsi (DAN), Disjungsi (ATAU)

p ∧ q (p DAN q): benar hanya jika keduanya benar.
p ∨ q (p ATAU q): benar jika salah satu atau keduanya benar.
p: "2 adalah prima" (B)
q: "3 adalah genap" (S)

p ∧ q: "2 prima DAN 3 genap" → Salah
p ∨ q: "2 prima ATAU 3 genap" → Benar

Implikasi (Jika-Maka)

p → q (jika p maka q).

Implikasi hanya salah jika p benar tapi q salah. Selain itu semua benar.

Tabel kebenaran p → q:

p = B, q = B → p → q = B
p = B, q = S → p → q = S
p = S, q = B → p → q = B
p = S, q = S → p → q = B

Dua baris terakhir mungkin mengejutkan: dari premis yang salah, implikasi dianggap benar (disebut "vacuous truth").

Contoh: "Jika hari hujan, maka jalan basah."

  • Hujan dan jalan basah → benar.
  • Hujan tapi jalan tidak basah → salah (melanggar).
  • Tidak hujan dan jalan basah (mungkin disiram) → tidak melanggar pernyataan → dianggap benar.
  • Tidak hujan dan jalan tidak basah → juga benar.

Konvers, Invers, Kontrapositif

Dari implikasi p → q, kita bisa membentuk:

Konvers: q → p
Invers: ¬p → ¬q
Kontrapositif: ¬q → ¬p   (ekuivalen dengan p → q!)
p: "Hari hujan."
q: "Jalan basah."

Implikasi: Jika hujan, maka jalan basah. (p → q)
Konvers: Jika jalan basah, maka hujan. (q → p) ← belum tentu benar!
Invers: Jika tidak hujan, maka jalan tidak basah. (¬p → ¬q) ← juga belum tentu benar!
Kontrapositif: Jika jalan tidak basah, maka tidak hujan. (¬q → ¬p) ← pasti benar!
💡 Kunci: Suatu implikasi selalu ekuivalen dengan kontrapositif-nya. Ini adalah senjata penting dalam pembuktian matematika.

5. Pembuktian Dasar — Meyakinkan Bahwa Sesuatu Itu Benar

Dalam matematika, pembuktian adalah argumen logis yang menunjukkan bahwa suatu pernyataan pasti benar, tanpa perlu mencoba semua kemungkinan.

Metode 1: Pembuktian Langsung

Mulai dari premis (yang diketahui), lalu dengan langkah-langkah logis sampai ke kesimpulan.

Buktikan: Jika n bilangan genap, maka n² juga genap.

Bukti:
Misalkan n genap, maka n = 2k untuk suatu bilangan bulat k.
n² = (2k)² = 4k² = 2(2k²)
Karena 2k² adalah bilangan bulat, maka n² adalah kelipatan 2 → genap. ✅

Metode 2: Pembuktian dengan Kontrapositif

Untuk membuktikan p → q, kita bisa membuktikan kontrapositifnya: ¬q → ¬p.

Buktikan: Jika n² ganjil, maka n ganjil.

Bukti dengan kontrapositif:
Kontrapositif: Jika n tidak ganjil (genap), maka n² tidak ganjil (genap).
Ini sudah kita buktikan di atas: n genap → n² genap. ✅

Metode 3: Pembuktian dengan Kontradiksi

Asumsikan yang mau dibuktikan itu salah, lalu tunjukkan bahwa asumsi itu menghasilkan kontradiksi (hal yang mustahil). Karena menghasilkan kontradiksi, maka asumsi pasti salah, artinya pernyataan aslinya benar.

Buktikan: √2 adalah bilangan irasional.

Bukti dengan kontradiksi:
Asumsikan √2 rasional, maka √2 = p/q dengan p dan q bilangan bulat yang sudah disederhanakan (FPB p,q = 1).

√2 = p/q
2 = p²/q²
p² = 2q² → p² genap → p genap → p = 2k

(2k)² = 2q²
4k² = 2q²
2k² = q² → q² genap → q genap.

p dan q keduanya genap → FPB(p,q) ≠ 1.
Ini kontradiksi dengan asumsi bahwa p/q sudah disederhanakan.
Jadi, asumsi √2 rasional salah. √2 adalah irasional. ✅

Ini adalah bukti klasik yang terkenal dari zaman Yunani kuno!

6. Notasi Sigma (Σ) — Menulis Penjumlahan Panjang dengan Ringkas

Bayangkan kamu harus menulis: 1 + 2 + 3 + 4 + ... + 100. Itu panjang dan tidak praktis.

Dengan notasi sigma, cukup tulis:

Σ (dari i=1 sampai n) dari aᵢ = a₁ + a₂ + a₃ + ... + aₙ

Dibaca: "Jumlah dari aᵢ untuk i dari 1 sampai n".

Komponen notasi sigma:

  • Σ (huruf Yunani sigma kapital) = simbol penjumlahan.
  • i = indeks (bisa pakai huruf apa saja: i, j, k, n).
  • Batas bawah (i = 1) = nilai awal indeks.
  • Batas atas (n) = nilai akhir indeks.
  • aᵢ = suku yang dijumlahkan (bisa berupa rumus dalam i).

Contoh-Contoh Notasi Sigma

Contoh 1:
Σ(i dari 1 sampai 5) i = 1 + 2 + 3 + 4 + 5 = 15

Contoh 2:
Σ(i dari 1 sampai 4) i² = 1² + 2² + 3² + 4² = 1 + 4 + 9 + 16 = 30

Contoh 3:
Σ(i dari 1 sampai 3) (2i + 1) = (2·1+1) + (2·2+1) + (2·3+1) = 3 + 5 + 7 = 15

Contoh 4:
Σ(k dari 1 sampai 100) k = 1 + 2 + ... + 100 = 5050

Sifat-Sifat Notasi Sigma

1. Σ c = c × n (c konstanta, dijumlahkan n kali)
2. Σ c·aᵢ = c · Σ aᵢ
3. Σ (aᵢ + bᵢ) = Σ aᵢ + Σ bᵢ
Σ(i=1 sampai 5) 3 = 3 + 3 + 3 + 3 + 3 = 15 = 3 × 5 ✅

Σ(i=1 sampai 4) 2i = 2 × Σ i = 2 × (1+2+3+4) = 2 × 10 = 20 ✅

Rumus-Rumus Penting Terkait Sigma

Σ(i=1 sampai n) i = n(n+1)/2
Σ(i=1 sampai n) i² = n(n+1)(2n+1)/6
Σ(i=1 sampai n) i³ = [n(n+1)/2]²
Σ(i=1 sampai 100) i = 100×101/2 = 5050

Σ(i=1 sampai 10) i² = 10×11×21/6 = 385

📝 Ringkasan Cepat

  • Nilai mutlak: jarak dari nol. |x| = x jika x≥0, -x jika x<0.
  • Interval: [a,b] tertutup, (a,b) terbuka, [a,∞) tak hingga.
  • Himpunan: ∩ irisan, ∪ gabungan, ⊆ subset, ∈ anggota.
  • Logika: ∧ (DAN), ∨ (ATAU), → (implikasi), ¬ (negasi).
  • Kontrapositif p→q adalah ¬q→¬p (ekuivalen).
  • Pembuktian: langsung, kontrapositif, kontradiksi.
  • Σ aᵢ: jumlah aᵢ dari i=1 sampai n.
  • Σ i = n(n+1)/2.

✨ Tips & Trik

  • Nilai mutlak: selalu pikirkan sebagai "jarak". |x - a| = jarak x ke a.
  • Interval: biasakan menggambar garis bilangan. Ini mencegah kesalahan.
  • Himpunan: simbol ∩ mirip "dan" (n dari irisan, n dari and). Simbol ∪ mirip "atau" (u dari union, u dari atau versi inggris? tidak juga sih, tapi bisa jadi jembatan keledai).
  • Implikasi vs Konvers: Jangan tertukar. "Jika hujan maka basah" tidak sama dengan "jika basah maka hujan".
  • Kontrapositif: senjata ampuh saat pembuktian langsung sulit.
  • Notasi sigma: latih mengubah dari bentuk sigma ke penjumlahan biasa dan sebaliknya. Ini akan sangat terpakai di statistika dan kalkulus.
