1. Apa Itu Matriks? — Definisi dan Notasi

Matriks adalah susunan bilangan (atau simbol) berbentuk persegi panjang yang diatur dalam baris dan kolom.

Matriks A berordo m × n memiliki m baris dan n kolom.

Elemen pada baris ke-i, kolom ke-j ditulis aᵢⱼ (atau Aᵢⱼ).
A = [ 2 5 7 ] → matriks 2×3 (2 baris, 3 kolom)
    [ 1 3 4 ]

a₁₁ = 2 (baris 1, kolom 1)
a₁₂ = 5 (baris 1, kolom 2)
a₂₃ = 4 (baris 2, kolom 3)

Mengapa matriks penting? Karena ia bisa merepresentasikan banyak hal sekaligus: sistem persamaan, transformasi geometri, data tabel, graf, dan masih banyak lagi. Matriks mengubah masalah besar menjadi operasi tabel yang rapi.

Ordo Matriks

Ordo adalah "ukuran" matriks, ditulis m × n (dibaca "m kali n").

Matriks 3×2: 3 baris, 2 kolom.
Matriks 1×4: matriks baris (hanya 1 baris).
Matriks 3×1: matriks kolom (hanya 1 kolom).
Matriks n×n: matriks persegi (square matrix) — jumlah baris = jumlah kolom.

2. Jenis-Jenis Matriks Khusus

Matriks Persegi: jumlah baris = jumlah kolom (n×n).

Matriks Diagonal: matriks persegi yang semua elemen di luar diagonal utama = 0.

Matriks Identitas (I): matriks diagonal dengan semua diagonal utama = 1.
I₂ = [1 0; 0 1], I₃ = [1 0 0; 0 1 0; 0 0 1].

Matriks Nol (O): semua elemen = 0.

Matriks Segitiga Atas: semua elemen di bawah diagonal utama = 0.

Matriks Segitiga Bawah: semua elemen di atas diagonal utama = 0.

Matriks Simetris: A = Aᵀ (elemen aᵢⱼ = aⱼᵢ).

Mengapa matriks identitas istimewa?

I dalam matriks seperti angka 1 dalam perkalian biasa: A × I = A dan I × A = A untuk setiap matriks A yang ordonya cocok. Mengalikan dengan I tidak mengubah matriks.

3. Operasi Dasar Matriks — Penjumlahan dan Pengurangan

Dua matriks bisa dijumlahkan atau dikurangkan hanya jika ordonya sama.

(A ± B)ᵢⱼ = aᵢⱼ ± bᵢⱼ

Jumlahkan/kurangkan elemen yang posisinya sama.
A = [3 1]   B = [2 4]
    [5 2]       [1 3]

A + B = [3+2   1+4] = [5   5]
        [5+1   2+3]    [6   5]

A - B = [3-2   1-4] = [1   -3]
        [5-1   2-3]    [4   -1]

Sifat penjumlahan matriks:

  • Komutatif: A + B = B + A
  • Asosiatif: (A + B) + C = A + (B + C)
  • Elemen identitas: A + O = A (O adalah matriks nol)

4. Perkalian Skalar — Mengalikan Matriks dengan Angka

(kA)ᵢⱼ = k × aᵢⱼ

Setiap elemen dikalikan dengan k.
A = [2 3]
    [4 1]

3A = [3×2   3×3] = [6   9]
     [3×4   3×1]    [12   3]

Sifat perkalian skalar:

  • k(A + B) = kA + kB (distributif)
  • (k + m)A = kA + mA
  • k(mA) = (km)A

5. Perkalian Dua Matriks — Operasi yang Paling Penting

Ini adalah operasi matriks yang paling sering dipakai dan paling sering salah. Pahami baik-baik.

Syarat: A (m×n) × B (p×q) bisa dikalikan jika n = p.
Jumlah kolom A harus sama dengan jumlah baris B.

Hasilnya: matriks berordo m × q.

Cara menghitung:
(A × B)ᵢⱼ = jumlah dari (baris ke-i A) × (kolom ke-j B)
= aᵢ₁b₁ⱼ + aᵢ₂b₂ⱼ + ... + aᵢₙbₙⱼ

Intuisi: Elemen di posisi (i,j) hasil perkalian adalah perkalian titik (dot product) antara baris ke-i dari A dan kolom ke-j dari B.

Contoh 1: (2×2) × (2×2)

A = [1 2]   B = [5 6]
    [3 4]       [7 8]

A × B = [1×5+2×7   1×6+2×8] = [5+14   6+16] = [19   22]
        [3×5+4×7   3×6+4×8]    [15+28   18+32]    [43   50]

Contoh 2: (2×3) × (3×2) → hasil (2×2)

A = [1 2 3]   B = [1 2]
    [4 5 6]       [3 4]
                      [5 6]

A × B = [1×1+2×3+3×5   1×2+2×4+3×6] = [1+6+15   2+8+18] = [22   28]
        [4×1+5×3+6×5   4×2+5×4+6×6]    [4+15+30   8+20+36]    [49   64]

Mengapa Perkalian Matriks Tidak Komutatif?

A × B ≠ B × A (pada umumnya).

Bahkan, bisa jadi A×B terdefinisi tapi B×A tidak terdefinisi!
A = [1 2]   B = [5 6]
    [3 4]       [7 8]

A × B = [19 22]
        [43 50]

B × A = [5×1+6×3   5×2+6×4] = [23   34]
        [7×1+8×3   7×2+8×4]    [31   46]

A × B ≠ B × A   (hasilnya berbeda!)

Kapan perkalian matriks komutatif? Kasus khusus: A × I = I × A. Juga untuk matriks diagonal dengan ukuran sama. Tapi secara umum, jangan berasumsi komutatif.

Sifat Perkalian Matriks

1. Asosiatif: (AB)C = A(BC)
2. Distributif: A(B + C) = AB + AC
3. Tidak komutatif: AB ≠ BA (umumnya)
4. Identitas: AI = IA = A

6. Transpose Matriks — Tukar Baris dan Kolom

Transpose dari A, ditulis Aᵀ (atau A'), didapat dengan menukar baris dan kolom.

(Aᵀ)ᵢⱼ = aⱼᵢ
A = [1 2 3]
    [4 5 6]

Aᵀ = [1 4]
     [2 5]
     [3 6]

Baris 1 A → kolom 1 Aᵀ. Baris 2 A → kolom 2 Aᵀ.

Sifat-Sifat Transpose

1. (Aᵀ)ᵀ = A
2. (A + B)ᵀ = Aᵀ + Bᵀ
3. (AB)ᵀ = BᵀAᵀ — perhatikan urutannya dibalik!
4. (kA)ᵀ = kAᵀ

Mengapa (AB)ᵀ = BᵀAᵀ?

A = [1 2]   B = [3]
    [3 4]       [5]

AB = [1×3+2×5] = [13]
     [3×3+4×5]    [29]

(AB)ᵀ = [13 29] (matriks 1×2)

BᵀAᵀ = [3 5] × [1 3] = [3×1+5×2   3×3+5×4] = [13 29] ✅
                [2 4]

7. Determinan Matriks 2×2 dan 3×3

Determinan adalah sebuah angka skalar yang dihitung dari matriks persegi. Determinan menentukan banyak sifat penting matriks, termasuk apakah matriks punya invers.

Determinan matriks A ditulis det(A) atau |A|.

Determinan 2×2 — Rumus Sederhana

A = [a b]
    [c d]

det(A) = ad - bc

Mengapa ad - bc?

Determinan bisa dipahami sebagai faktor skala luas. Jika kamu punya persegi satuan (luas 1) lalu ditransformasi oleh matriks A, luas hasil transformasi = |det(A)|. Inilah makna geometrisnya.

A = [3 2]
    [1 4]

det(A) = 3×4 - 2×1 = 12 - 2 = 10 ✅

Determinan 3×3 — Aturan Sarrus

Untuk matriks 3×3, determinan bisa dihitung dengan Aturan Sarrus:
Tulis ulang kolom 1 dan 2 di sebelah kanan, lalu jumlahkan diagonal utama dan kurangkan diagonal samping.
A = [1 2 3]
    [4 5 6]
    [7 8 9]

Tulis ulang kolom 1, 2:
1 2 3 | 1 2
4 5 6 | 4 5
7 8 9 | 7 8

Diagonal utama (↘): 1×5×9 + 2×6×7 + 3×4×8
= 45 + 84 + 96 = 225

Diagonal samping (↙): 3×5×7 + 1×6×8 + 2×4×9
= 105 + 48 + 72 = 225

det(A) = 225 - 225 = 0 ✅

Determinan 0 menandakan matriks singular (tidak punya invers).

Ekspansi Kofaktor (Laplace) — Untuk Determinan Lebih Besar

Untuk matriks 3×3 atau lebih besar, kita bisa menggunakan ekspansi kofaktor (ekspansi Laplace). Pilih satu baris atau kolom, lalu hitung minor dan kofaktornya.

Minor dan Kofaktor

Minor Mᵢⱼ: determinan matriks setelah menghapus baris i dan kolom j.

Kofaktor Cᵢⱼ: Cᵢⱼ = (-1)ⁱ⁺ʲ × Mᵢⱼ
A = [2 3 1]
    [4 5 6]
    [7 8 9]

Minor M₁₁: hapus baris 1, kolom 1 → [5 6; 8 9] → det = 5×9 - 6×8 = 45-48 = -3.
Kofaktor C₁₁: (-1)¹⁺¹ × (-3) = 1 × (-3) = -3.

Determinan = jumlah (elemen baris/kolom × kofaktornya). Untuk ekspansi di baris pertama:

det(A) = a₁₁C₁₁ + a₁₂C₁₂ + a₁₃C₁₃
= 2(-3) + 3(-C₁₂) + 1(C₁₃) ... (lanjutkan menghitung C₁₂ dan C₁₃)

8. Sifat-Sifat Determinan

1. det(Aᵀ) = det(A) — transpose tidak mengubah determinan.
2. det(AB) = det(A) × det(B) — determinan perkalian = perkalian determinan.
3. det(kA) = kⁿ det(A) — untuk matriks n×n, setiap baris dikali k.
4. det(I) = 1 — determinan matriks identitas = 1.
5. det(A⁻¹) = 1/det(A) — determinan invers = kebalikan determinan.
6. Jika ada baris/kolom nol → det = 0.
7. Jika ada baris/kolom yang kelipatan → det = 0.
8. Menukar dua baris → det berubah tanda.
9. Menambah kelipatan satu baris ke baris lain → det TETAP.

Sifat 9 sangat penting dalam menghitung determinan dengan OBE: kita bisa menyederhanakan matriks menjadi segitiga tanpa mengubah determinan (asalkan tidak menukar baris).

9. Invers Matriks — "Kebalikan" Matriks

Invers matriks (A⁻¹) adalah matriks yang jika dikalikan dengan A menghasilkan matriks identitas I.

A × A⁻¹ = A⁻¹ × A = I

Syarat: A harus persegi dan determinannya ≠ 0 (non-singular).

Mencari Invers 2×2 — Rumus Langsung

Untuk A = [a b]:
          [c d]

A⁻¹ = 1/(ad-bc) × [d   -b]
                   [-c   a]
A = [3 2]
    [1 4]

det(A) = 10

A⁻¹ = 1/10 × [4   -2] = [0,4   -0,2]
             [-1   3]    [-0,1   0,3]

Verifikasi: A × A⁻¹ = [3(0,4)+2(-0,1)   3(-0,2)+2(0,3)] = [1 0] = I ✅
                         [1(0,4)+4(-0,1)   1(-0,2)+4(0,3)]    [0 1]

Mencari Invers 3×3 — Metode Adjoin

Langkah-langkah:
1. Hitung det(A). Kalau 0, berhenti (tidak ada invers).
2. Hitung matriks kofaktor (semua Cᵢⱼ).
3. Transpose matriks kofaktor → dapat Adjoin(A).
4. A⁻¹ = Adjoin(A) / det(A).

10. Operasi Baris Elementer (OBE) — Alat Paling Powerful

Operasi Baris Elementer (OBE) adalah tiga operasi yang bisa dilakukan pada baris matriks tanpa mengubah solusi sistem persamaan.

Tiga jenis OBE:

1. Menukar dua baris: Rᵢ ↔ Rⱼ
2. Mengalikan satu baris dengan konstanta ≠ 0: kRᵢ → Rᵢ
3. Menambah kelipatan satu baris ke baris lain: Rᵢ + kRⱼ → Rᵢ

Mengapa OBE tidak mengubah solusi SPL? Karena setiap OBE ekuivalen dengan operasi aljabar yang sah pada persamaan: menukar urutan persamaan, mengalikan persamaan dengan konstanta, atau menjumlahkan dua persamaan.

Eliminasi Gauss — Membentuk Eselon Baris

Tujuan: mengubah matriks menjadi bentuk eselon baris (segitiga atas) menggunakan OBE.

Ciri bentuk eselon baris:
- Baris nol (jika ada) di paling bawah.
- Leading entry (angka pertama bukan nol) di baris bawah selalu di kanan leading entry di atasnya.

Eliminasi Gauss-Jordan — Membentuk Eselon Baris Tereduksi

Tujuan: mengubah matriks menjadi bentuk eselon baris tereduksi.

Ciri tambahan:
- Setiap leading entry = 1.
- Setiap leading entry adalah satu-satunya elemen bukan nol di kolomnya.

11. Mencari Invers dengan OBE — Metode [A|I]

Ini adalah metode paling efisien untuk mencari invers matriks besar.

Langkah:
1. Tulis [A | I] (matriks A di kiri, identitas di kanan).
2. Lakukan OBE sampai sisi kiri menjadi I.
3. Sisi kanan sekarang adalah A⁻¹.
A = [2 1]
    [5 3]

[A|I] = [2 1 | 1 0]
        [5 3 | 0 1]

R₁/2 → R₁: [1 0,5 | 0,5 0]
             [5   3   | 0    1]

R₂ - 5R₁ → R₂: [1 0,5 | 0,5 0]
                [0 0,5 | -2,5 1]

2R₂ → R₂: [1 0,5 | 0,5   0]
          [0   1   | -5    2]

R₁ - 0,5R₂ → R₁: [1 0 | 3   -1]
                  [0 1 | -5   2]

A⁻¹ = [3   -1]
      [-5   2] ✅

12. Rank Matriks — Menentukan Banyaknya Solusi SPL

Rank matriks adalah jumlah maksimum baris tak nol dalam bentuk eselon barisnya.

Rank menentukan apakah SPL punya solusi tunggal, banyak, atau tidak ada.
Untuk SPL AX = B dengan n variabel:

rank(A) = rank([A|B]) = n → solusi tunggal.
rank(A) = rank([A|B]) < n → solusi banyak (tak hingga).
rank(A) < rank([A|B]) → tidak ada solusi (inkonsisten).

13. Penerapan Matriks ke Sistem Persamaan Linear

SPL bisa ditulis sebagai persamaan matriks AX = B.

2x + 3y = 8
5x - y = 3

Bentuk matriks:
[2 3] [x] = [8]
[5 -1] [y]    [3]

A = [2 3], X = [x], B = [8]
    [5 -1]      [y]      [3]

Menyelesaikan dengan Invers

Jika det(A) ≠ 0: X = A⁻¹B

Aturan Cramer

xᵢ = det(Aᵢ) / det(A)

di mana Aᵢ adalah matriks A dengan kolom ke-i diganti B.
Untuk sistem di atas: det(A) = 2(-1) - 3(5) = -2 - 15 = -17.

x = det([8 3; 3 -1])/det(A) = (8(-1) - 3(3))/(-17) = (-8-9)/(-17) = 1

y = det([2 8; 5 3])/det(A) = (2(3) - 8(5))/(-17) = (6-40)/(-17) = 2

Solusi: x = 1, y = 2 ✅

14. Persamaan Matriks — Mencari X dari AX = B atau XA = B

AX = B → X = A⁻¹B (kalikan A⁻¹ dari kiri).

XA = B → X = BA⁻¹ (kalikan A⁻¹ dari kanan).

AXB = C → X = A⁻¹CB⁻¹.

Hati-hati: Karena perkalian matriks tidak komutatif, posisi A⁻¹ sangat penting. Di AX = B, A⁻¹ harus di kiri. Di XA = B, A⁻¹ harus di kanan.

15. Trace Matriks — Jumlah Diagonal Utama

Trace(A) = tr(A) = jumlah semua elemen diagonal utama.

Untuk A = [aᵢⱼ]ₙₓₙ: tr(A) = a₁₁ + a₂₂ + ... + aₙₙ
A = [2 3 1]
    [4 5 6]
    [7 8 9]

tr(A) = 2 + 5 + 9 = 16

Sifat-Sifat Trace

1. tr(A + B) = tr(A) + tr(B)
2. tr(kA) = k·tr(A)
3. tr(AB) = tr(BA) — sangat berguna sebagai trik!
4. tr(Aᵀ) = tr(A)

16. Determinan Matriks n×n — Ekspansi Kofaktor

Untuk matriks 4×4 atau lebih besar, kita menggunakan ekspansi kofaktor. Pilih satu baris atau kolom (sebaiknya yang paling banyak nolnya), lalu hitung determinan secara rekursif.

det(A) = Σ (-1)ⁱ⁺ʲ aᵢⱼ Mᵢⱼ (jumlah sepanjang satu baris/kolom)

17. Nilai Eigen dan Vektor Eigen — Pengantar

Ini adalah puncak dari teori matriks, biasanya baru dipelajari di universitas.

Jika Av = λv untuk suatu skalar λ dan vektor tak nol v, maka:

λ = nilai eigen (eigenvalue)
v = vektor eigen (eigenvector)

Makna: Ketika matriks A mengalikan vektor v, hasilnya adalah vektor yang sama (v) hanya saja diperpanjang/diperpendek oleh faktor λ. Arahnya tidak berubah!

Mencari Nilai Eigen

Persamaan karakteristik: det(A - λI) = 0

Selesaikan untuk λ. Ini adalah polinomial dalam λ.
A = [2 1]
    [1 2]

A - λI = [2-λ   1]
         [1   2-λ]

det(A - λI) = (2-λ)(2-λ) - 1 = λ² - 4λ + 3 = 0
(λ - 1)(λ - 3) = 0

Nilai eigen: λ = 1 dan λ = 3 ✅

Nilai eigen punya banyak aplikasi: analisis getaran, PageRank Google, pengenalan wajah (eigenface), mekanika kuantum, dan masih banyak lagi.

📝 Ringkasan Cepat

  • Matriks: susunan bilangan dalam baris dan kolom. Ordo m×n.
  • Penjumlahan: syarat ordo sama. Elemen seposisi dijumlahkan.
  • Perkalian matriks: A(m×n) × B(n×p) = C(m×p). Tidak komutatif.
  • Transpose: tukar baris dan kolom. (AB)ᵀ = BᵀAᵀ.
  • Determinan 2×2: ad - bc. Determinan 3×3: Sarrus atau kofaktor.
  • det(AB) = det(A)det(B). det(A⁻¹) = 1/det(A).
  • Invers 2×2: 1/det × adjoin. Invers n×n: [A|I] → OBE → [I|A⁻¹].
  • OBE: tukar baris, kali baris dengan k, tambah kelipatan baris lain.
  • Eliminasi Gauss: bentuk eselon baris. Gauss-Jordan: bentuk tereduksi.
  • Rank: jumlah baris tak nol di eselon baris. Menentukan solusi SPL.
  • SPL: AX = B → X = A⁻¹B. Cramer: xᵢ = det(Aᵢ)/det(A).
  • Trace: jumlah diagonal utama. tr(AB) = tr(BA).
  • Nilai eigen λ: det(A - λI) = 0. Vektor eigen v: Av = λv.

✨ Tips & Trik

  • Cek ordo sebelum menjumlahkan/mengalikan. Kesalahan paling umum adalah mengalikan dua matriks yang ordonya tidak cocok.
  • Perkalian matriks: baris × kolom. Baris dari matriks kiri dikalikan dengan kolom dari matriks kanan. Jangan terbalik.
  • (AB)ᵀ = BᵀAᵀ, bukan AᵀBᵀ. Urutannya dibalik. Ini sering jadi jebakan.
  • Determinan 3×3 pakai Sarrus untuk cepat. Tapi untuk 4×4 ke atas, pakai ekspansi kofaktor dengan memilih baris/kolom yang paling banyak nol.
  • Kalau det = 0, matriks singular → tidak punya invers. Jangan lanjutkan mencari invers, karena memang tidak ada.
  • Untuk invers matriks besar, pakai OBE [A|I]. Metode adjoin terlalu lambat untuk matriks 4×4 ke atas.
  • Rank menentukan jenis solusi SPL. Rank(A) = rank([A|B]) = n → solusi tunggal. Rank lebih kecil → banyak solusi. Rank berbeda → tidak ada solusi.
  • Persamaan matriks: perhatikan posisi A⁻¹. AX = B → X = A⁻¹B (A⁻¹ di kiri). XA = B → X = BA⁻¹ (A⁻¹ di kanan).
  • Trace itu sederhana tapi powerful. tr(AB) = tr(BA) sering jadi trik cepat untuk membuktikan sesuatu tanpa menghitung penuh.
  • Nilai eigen = akar polinomial karakteristik. det(A - λI) = 0. Untuk matriks 2×2, ini persamaan kuadrat. Untuk 3×3, ini persamaan kubik.
