Part 6 • Jenis Fungsi, Grafik & Transformasi

📖 Pendahuluan

Di Bab 5, kita sudah mempelajari konsep dasar fungsi: definisi, domain, range, piecewise, komposisi, dan invers. Sekarang kita akan menjelajahi jenis-jenis fungsi yang paling penting dalam matematika.

Setiap jenis fungsi punya "kepribadian" unik: bentuk grafik yang khas, sifat-sifat khusus, dan cara penanganan yang berbeda. Menguasai mereka berarti kamu bisa mengenali pola, memprediksi perilaku, dan menyelesaikan masalah dengan lebih cepat.

Kita akan membahas enam jenis fungsi: kuadrat, rasional, eksponen, logaritma, nilai mutlak, dan grafiknya. Setelah itu, kita akan mempelajari transformasi grafik: bagaimana menggeser, meregangkan, dan memantulkan grafik fungsi.

Tujuan Bab
  • Memahami karakteristik fungsi kuadrat dan grafik parabolanya.
  • Menguasai fungsi rasional dan asimtotnya.
  • Memahami fungsi eksponen, pertumbuhan, dan peluruhan.
  • Memahami fungsi logaritma sebagai invers eksponen.
  • Menguasai fungsi nilai mutlak dan grafiknya yang berbentuk V.
  • Mampu menggambar grafik fungsi dengan transformasi (translasi, refleksi, dilatasi).

1. Fungsi Kuadrat — Si Parabola

Fungsi kuadrat adalah fungsi dengan bentuk umum:

f(x) = ax² + bx + c,   a ≠ 0

Grafiknya berbentuk parabola.

Mengapa disebut parabola? Dari bahasa Yunani "parabole" yang berarti "melempar berdampingan". Lintasan benda yang dilempar ke atas (mengabaikan hambatan udara) berbentuk parabola. Matematika dan fisika bertemu di sini.

Karakteristik Penting Parabola

1. Arah bukaan: a > 0 → terbuka ke atas (minimum). a < 0 → terbuka ke bawah (maksimum).

2. Titik puncak (vertex): (-b/2a, f(-b/2a))

3. Sumbu simetri: x = -b/2a (garis vertikal yang membagi parabola menjadi dua sama persis).

4. Titik potong sumbu Y: (0, c)

5. Titik potong sumbu X: akar-akar dari ax² + bx + c = 0 (jika D ≥ 0).

Mencari Titik Puncak — Pembuktian

Mengapa x = -b/2a adalah sumbu simetri?

Fungsi kuadrat bisa ditulis dalam bentuk kuadrat sempurna:

f(x) = ax² + bx + c
= a(x² + (b/a)x) + c
= a(x² + (b/a)x + (b/2a)² - (b/2a)²) + c
= a((x + b/2a)² - b²/4a²) + c
= a(x + b/2a)² - b²/4a + c
= a(x + b/2a)² + (4ac - b²)/4a

Suku a(x + b/2a)² menentukan pergeseran horizontal.
Karena (x + b/2a)² ≥ 0, nilai minimum/maksimum terjadi saat
x + b/2a = 0 → x = -b/2a ✅

Contoh 1: Analisis Lengkap

f(x) = 2x² - 8x + 5

a = 2 > 0 → parabola terbuka ke atas (punya minimum).

Sumbu simetri: x = -(-8)/(2·2) = 8/4 = 2

Titik puncak: f(2) = 2(4) - 8(2) + 5 = 8 - 16 + 5 = -3
Puncak: (2, -3)

Titik potong Y: (0, 5)

Titik potong X: 2x² - 8x + 5 = 0
D = 64 - 40 = 24 > 0 → dua akar real.
x = (8 ± √24)/4 = (8 ± 2√6)/4 = 2 ± √6/2
x₁ ≈ 3,22   x₂ ≈ 0,78

Range: [-3, ∞) (karena minimum -3)

Bentuk Umum vs Bentuk Puncak

Bentuk umum: f(x) = ax² + bx + c (baik untuk mencari titik potong sumbu Y dan akar).

Bentuk puncak: f(x) = a(x - h)² + k, di mana (h, k) adalah titik puncak.
(Baik untuk menggambar grafik dan melihat pergeseran.)

Mengubah bentuk umum ke bentuk puncak: gunakan teknik melengkapkan kuadrat yang sudah kita pelajari di Aljabar 2.

f(x) = 2x² - 8x + 5

= 2(x² - 4x) + 5
= 2(x² - 4x + 4 - 4) + 5
= 2((x - 2)² - 4) + 5
= 2(x - 2)² - 8 + 5
= 2(x - 2)² - 3

Puncak: (2, -3) ✅

2. Fungsi Rasional — Si Asimtot

Fungsi rasional adalah fungsi berbentuk:

f(x) = P(x)/Q(x)

di mana P(x) dan Q(x) adalah polinomial, dan Q(x) ≠ 0.

Yang membuat fungsi rasional istimewa adalah adanya asimtot — garis-garis yang didekati grafik tapi tidak pernah disentuh (atau disentuh di tak hingga).

Jenis-Jenis Asimtot

1. Asimtot Vertikal: terjadi di x = a jika penyebut = 0 di x = a.
Grafik meluncur ke ∞ atau -∞ saat mendekati x = a.

2. Asimtot Horizontal: terjadi jika x → ±∞.
Dilihat dari perbandingan derajat pembilang dan penyebut.

3. Asimtot Miring: terjadi jika derajat pembilang = derajat penyebut + 1.

Aturan Mencari Asimtot Horizontal

Misal f(x) = (axᵐ + ...)/(bxⁿ + ...), dengan a, b ≠ 0.

m < n: asimtot horizontal y = 0 (sumbu X).
m = n: asimtot horizontal y = a/b.
m > n: tidak ada asimtot horizontal (mungkin ada asimtot miring).

Mengapa aturan ini berlaku?

Untuk x yang sangat besar, suku dengan pangkat tertinggi mendominasi. Suku-suku lain bisa diabaikan. Jadi f(x) ≈ (axᵐ)/(bxⁿ) = (a/b)xᵐ⁻ⁿ. Kalau m < n, pangkatnya negatif → mendekati 0. Kalau m = n, mendekati a/b. Kalau m > n, nilainya membesar/mengecil tanpa batas.

Contoh 1: Asimtot Vertikal dan Horizontal

f(x) = (2x+1)/(x-3)

Domain: x ≠ 3

Asimtot vertikal: x = 3 (penyebut = 0).

Asimtot horizontal: derajat pembilang = derajat penyebut = 1.
y = 2/1 = 2

Range: y ≠ 2 (semua bilangan real kecuali 2, karena tidak ada x yang menghasilkan tepat 2).

Grafik: Dua cabang hiperbola, satu di kuadran kanan-atas, satu di kiri-bawah.

Contoh 2: Asimtot Miring

f(x) = (x² + 1)/(x - 1)

Derajat pembilang (2) = derajat penyebut (1) + 1 → ada asimtot miring.

Asimtot vertikal: x = 1

Lakukan pembagian polinomial:
x² + 1 dibagi x - 1 → hasil bagi = x + 1, sisa = 2.

f(x) = x + 1 + 2/(x-1)

Untuk x → ±∞, 2/(x-1) → 0, sehingga f(x) ≈ x + 1.

Asimtot miring: y = x + 1 ✅

Contoh 3: Grafik Rasional dengan Dua Asimtot Vertikal

f(x) = 1/(x² - 4) = 1/[(x+2)(x-2)]

Asimtot vertikal: x = -2 dan x = 2
Asimtot horizontal: derajat pembilang (0) < derajat penyebut (2) → y = 0

Grafik memiliki tiga cabang: kiri x=-2, antara -2 dan 2, kanan x=2.

3. Fungsi Eksponen — Pertumbuhan dan Peluruhan

Fungsi eksponen adalah fungsi dengan variabel di posisi pangkat.

f(x) = a·bˣ

a = nilai awal (saat x = 0, f(0) = a).
b = basis (b > 0, b ≠ 1).

Dua jenis utama:

b > 1: Pertumbuhan eksponensial. Grafik naik dari kiri ke kanan.
Contoh: populasi bakteri, bunga majemuk, pertumbuhan investasi.

0 < b < 1: Peluruhan eksponensial. Grafik turun dari kiri ke kanan.
Contoh: peluruhan radioaktif, penurunan suhu, depresiasi aset.

Karakteristik Fungsi Eksponen f(x) = a·bˣ

Domain: ℝ (semua bilangan real).
Range: (0, ∞) jika a > 0; (-∞, 0) jika a < 0.
Asimtot horizontal: y = 0 (sumbu X). Grafik tidak pernah menyentuh sumbu X.
Titik potong Y: (0, a).
Tidak ada titik potong X (kecuali jika digeser).

Mengapa range hanya bilangan positif (untuk a > 0)?

Bilangan positif dipangkatkan berapapun (positif, nol, negatif) hasilnya selalu positif. 2³ = 8, 2⁰ = 1, 2⁻³ = 1/8. Tidak mungkin menghasilkan 0 atau negatif. Itulah kenapa asimtotnya y = 0 — grafik mendekati nol tapi tidak pernah menyentuhnya.

Contoh 1: Pertumbuhan Eksponensial

f(x) = 2ˣ

f(0) = 1, f(1) = 2, f(2) = 4, f(3) = 8, f(4) = 16, ...

Ciri khas: setiap x bertambah 1, nilai fungsi dilipatgandakan (dikali b).

Domain: ℝ
Range: (0, ∞)
Asimtot: y = 0

Contoh 2: Peluruhan Eksponensial

f(x) = 100·(1/2)ˣ

f(0) = 100, f(1) = 50, f(2) = 25, f(3) = 12,5, ...

Ciri khas: setiap x bertambah 1, nilai fungsi dibagi dua (dikali 1/b).

Ini disebut waktu paruh (half-life) dalam peluruhan radioaktif.

Bilangan e — Basis Istimewa

Ada satu basis yang sangat istimewa dalam matematika: e ≈ 2,71828.... Fungsi f(x) = eˣ disebut fungsi eksponensial natural. Keistimewaannya: laju pertumbuhannya sama dengan nilainya sendiri. Ini akan dipelajari lebih dalam di kalkulus.

4. Fungsi Logaritma — Invers dari Eksponen

Fungsi logaritma adalah invers dari fungsi eksponen.

y = ᵃlog x   ⇔   aʸ = x

a = basis (a > 0, a ≠ 1).
x = numerus (x > 0).

Kalau fungsi eksponen bertanya "berapa hasil dari a dipangkatkan x?", fungsi logaritma bertanya "x harus dipangkatkan berapa agar hasilnya a?"

Karakteristik Fungsi Logaritma f(x) = ᵃlog x

Domain: (0, ∞) — numerus harus positif.
Range: ℝ (semua bilangan real).
Asimtot vertikal: x = 0 (sumbu Y). Grafik tidak pernah menyentuh sumbu Y.
Titik potong X: (1, 0) — karena ᵃlog 1 = 0 untuk semua basis a.
Tidak ada titik potong Y (karena domain x > 0).

Mengapa ᵃlog 1 selalu = 0? Karena a⁰ = 1 untuk semua a ≠ 0. Logaritma menjawab: "a dipangkatkan berapa agar jadi 1?" Jawabannya selalu 0.

Dua Jenis Grafik Logaritma

a > 1: Grafik naik dari kiri ke kanan. Fungsi monoton naik.
0 < a < 1: Grafik turun dari kiri ke kanan. Fungsi monoton turun.

Contoh

f(x) = ²log x

f(1) = 0, f(2) = 1, f(4) = 2, f(8) = 3, f(1/2) = -1.

Domain: (0, ∞)
Range: ℝ
Asimtot: x = 0

Hubungan Grafik Eksponen dan Logaritma

Karena logaritma adalah invers eksponen, grafik y = ᵃlog x adalah pencerminan grafik y = aˣ terhadap garis y = x.

Titik (0, 1) pada y = 2ˣ → titik (1, 0) pada y = ²log x.
Titik (1, 2) pada y = 2ˣ → titik (2, 1) pada y = ²log x.
Titik (2, 4) pada y = 2ˣ → titik (4, 2) pada y = ²log x.

5. Fungsi Nilai Mutlak — Si Grafik V

Fungsi nilai mutlak adalah:

f(x) = |x| = { x,   jika x ≥ 0
              { -x, jika x < 0

Grafiknya berbentuk V dengan titik sudut di (0,0).

Karakteristik f(x) = |x|

Domain: ℝ
Range: [0, ∞)
Titik sudut: (0, 0) — titik di mana aturan berubah.
Simetri: fungsi genap (f(-x) = f(x)), grafik simetris terhadap sumbu Y.

Bentuk Umum: f(x) = a|x - h| + k

(h, k) = titik sudut.
a > 0: V terbuka ke atas. a < 0: V terbuka ke bawah.
|a| > 1: V lebih sempit (curam). 0 < |a| < 1: V lebih lebar (landai).

Contoh

f(x) = 2|x - 3| + 1

Titik sudut: (3, 1)
a = 2 > 0: V terbuka ke atas, lebih curam dari |x|.

f(3) = 1 (titik sudut)
f(4) = 2|1| + 1 = 3
f(5) = 2|2| + 1 = 5
f(2) = 2|-1| + 1 = 3

Range: [1, ∞)

6. Transformasi Grafik — Menggeser, Meregangkan, Memantulkan

Transformasi grafik adalah teknik mengubah grafik fungsi dasar menjadi grafik fungsi yang lebih kompleks. Dengan transformasi, kita bisa menggambar grafik tanpa harus menghitung banyak titik.

Ada tiga jenis transformasi dasar:

1. Translasi (pergeseran) — grafik digeser tanpa berubah bentuk.
2. Refleksi (pencerminan) — grafik dicerminkan terhadap sumbu.
3. Dilatasi (peregangan/penyempitan) — grafik diregangkan atau dimampatkan.

1. Translasi (Pergeseran)

Vertikal: f(x) + k
k > 0 → geser ke atas sejauh k.
k < 0 → geser ke bawah sejauh |k|.

Horizontal: f(x - h)
h > 0 → geser ke kanan sejauh h.
h < 0 → geser ke kiri sejauh |h|.

Mengapa f(x - h) bergeser ke kanan saat h positif?

Ini sering membingungkan. Logikanya: untuk mendapatkan nilai yang sama, input harus lebih besar. Misal f(x) = x² punya nilai 0 di x = 0. Untuk f(x - 3) = (x-3)², nilai 0 terjadi saat x - 3 = 0 → x = 3. Jadi grafik "terlambat" 3 satuan — efeknya bergeser ke kanan.

f(x) = x²

f(x) + 3 = x² + 3 → parabola naik 3 satuan.
f(x - 2) = (x-2)² → parabola geser kanan 2 satuan.
f(x + 1) - 4 = (x+1)² - 4 → parabola geser kiri 1, bawah 4.

2. Refleksi (Pencerminan)

Terhadap sumbu X: -f(x)
Setiap y berubah tanda → grafik dicerminkan terhadap sumbu X.

Terhadap sumbu Y: f(-x)
Setiap x berubah tanda → grafik dicerminkan terhadap sumbu Y.
f(x) = √x (hanya di x ≥ 0)

-f(x) = -√x → grafik akar dicerminkan ke bawah.
f(-x) = √(-x) → grafik akar dicerminkan ke kiri (domain jadi x ≤ 0).

3. Dilatasi (Peregangan/Penyempitan)

Vertikal: a·f(x)
|a| > 1 → grafik diregangkan vertikal (lebih curam).
0 < |a| < 1 → grafik dimampatkan vertikal (lebih landai).
a < 0 → ditambah refleksi terhadap sumbu X.

Horizontal: f(bx)
|b| > 1 → grafik dimampatkan horizontal (lebih sempit).
0 < |b| < 1 → grafik diregangkan horizontal (lebih lebar).
b < 0 → ditambah refleksi terhadap sumbu Y.

Perhatikan: efek horizontal "berlawanan" dengan intuisi. b > 1 membuat grafik lebih sempit, bukan lebih lebar. Karena input "dipercepat" — mencapai nilai yang sama dengan x yang lebih kecil.

f(x) = x²

2f(x) = 2x² → parabola lebih sempit/curam.
(1/2)f(x) = (1/2)x² → parabola lebih lebar/landai.
f(2x) = 4x² → parabola lebih sempit (input "dipercepat").
f(x/2) = x²/4 → parabola lebih lebar (input "diperlambat").

Menggambar Grafik dengan Transformasi — Urutan Operasi

Saat fungsi memiliki beberapa transformasi sekaligus, gunakan urutan ini:

Urutan standar:
1. Translasi horizontal (dalam kurung)
2. Dilatasi & Refleksi horizontal
3. Dilatasi & Refleksi vertikal
4. Translasi vertikal

Atau lebih praktis: ikuti aturan PEMDAS terbalik. Operasi yang paling "dekat" dengan x dikerjakan lebih dulu.

Contoh Lengkap

Gambarkan g(x) = -2(x + 3)² + 5 dari f(x) = x².

Langkah 1: f(x + 3) = (x+3)² → geser kiri 3.
Langkah 2: 2(x+3)² → regangkan vertikal 2 kali.
Langkah 3: -2(x+3)² → cerminkan terhadap sumbu X.
Langkah 4: -2(x+3)² + 5 → geser atas 5.

Hasil akhir: Parabola terbalik (∩), puncak di (-3, 5), lebih curam.

Transformasi pada Berbagai Jenis Fungsi

Fungsi Nilai Mutlak:
f(x) = |x| → titik sudut (0,0).
g(x) = |x - 2| + 3 → titik sudut (2, 3).

Fungsi Eksponen:
f(x) = 2ˣ → asimtot y = 0.
g(x) = 2ˣ⁺¹ - 3 → asimtot y = -3, geser kiri 1.

Fungsi Logaritma:
f(x) = ²log x → asimtot x = 0.
g(x) = ²log(x - 1) + 2 → asimtot x = 1, geser atas 2.

📝 Ringkasan Cepat

  • Fungsi Kuadrat: parabola. a > 0 → ∪. a < 0 → ∩. Puncak di x = -b/2a.
  • Fungsi Rasional: asimtot vertikal (penyebut = 0), asimtot horizontal (bandingkan derajat).
  • Fungsi Eksponen: domain ℝ, range (0,∞), asimtot y = 0. b > 1 naik, 0 < b < 1 turun.
  • Fungsi Logaritma: domain (0,∞), range ℝ, asimtot x = 0. Invers dari eksponen.
  • Fungsi Nilai Mutlak: grafik V. f(x) = a|x-h|+k, titik sudut (h,k).
  • Translasi: f(x)+k (vertikal), f(x-h) (horizontal, h>0 → kanan).
  • Refleksi: -f(x) (terhadap X), f(-x) (terhadap Y).
  • Dilatasi: a·f(x) (vertikal), f(bx) (horizontal, |b|>1 → sempit).

✨ Tips & Trik

  • Kenali bentuk dasar. Sebelum menganalisis fungsi kompleks, kenali dulu fungsi dasarnya: x², 1/x, aˣ, log x, |x|. Transformasi hanyalah modifikasi dari bentuk dasar.
  • Untuk parabola, cari puncak dulu. Dari puncak, grafik bisa digambar dengan cepat karena simetris.
  • Untuk fungsi rasional, cari asimtot dulu. Asimtot adalah "kerangka" grafik. Setelah asimtot ditemukan, tinggal tentukan di kuadran mana grafik berada.
  • Eksponen dan logaritma adalah pasangan invers. Grafik keduanya simetris terhadap y = x. Ini membantu mengecek apakah gambar grafikmu benar.
  • Transformasi horizontal terasa "terbalik". f(x+3) geser ke kiri. f(2x) malah menyempit. Jangan mengandalkan intuisi, pahami logikanya.
  • Gambar grafik dengan urutan yang benar. Translasi horizontal dulu, baru dilatasi/refleksi, terakhir translasi vertikal.
  • Verifikasi dengan satu titik. Setelah menggambar grafik hasil transformasi, uji satu titik untuk memastikan tidak ada kesalahan.
