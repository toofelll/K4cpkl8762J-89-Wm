Part 4 • Pertidaksamaan

📖 Pendahuluan

Di bab-bab sebelumnya, kita selalu berurusan dengan tanda = (sama dengan). Sekarang kita masuk ke dunia pertidaksamaan: mencari nilai-nilai yang membuat suatu ekspresi lebih besar, lebih kecil, atau kombinasinya dari ekspresi lain.

Simbol-simbol pertidaksamaan:

< : kurang dari
> : lebih dari
≤ : kurang dari atau sama dengan
≥ : lebih dari atau sama dengan

Perbedaan mendasar dengan persamaan:

  • Persamaan biasanya punya beberapa solusi titik (x = 2, x = -3).
  • Pertidaksamaan biasanya punya interval solusi (x > 5, -2 ≤ x < 3).
  • Mengali atau membagi dengan bilangan negatif membalik tanda. Ini jebakan paling umum!

Kita akan membahas pertidaksamaan secara bertahap: linear, kuadrat, rasional, nilai mutlak, eksponen, logaritma, dan sistem.

Tujuan Bab
  • Memahami prinsip dasar pertidaksamaan dan kapan tanda dibalik.
  • Mampu menyelesaikan pertidaksamaan linear dan kuadrat.
  • Menguasai metode garis bilangan dan uji titik.
  • Mampu menyelesaikan pertidaksamaan rasional.
  • Memahami pertidaksamaan nilai mutlak.
  • Mengenal pertidaksamaan eksponen dan logaritma.
  • Mampu menyelesaikan sistem pertidaksamaan.

1. Prinsip Dasar Pertidaksamaan

Aturan mainnya hampir sama dengan persamaan, tapi ada satu aturan tambahan yang krusial:

Menambah atau mengurangi bilangan yang sama di kedua ruas → tanda TETAP.

Mengali atau membagi dengan bilangan positif → tanda TETAP.

Mengali atau membagi dengan bilangan negatif → tanda DIBALIK.

Mengapa tanda harus dibalik saat dikali/dibagi negatif?

Bayangkan garis bilangan. Ambil contoh sederhana:

2 < 5   (pernyataan benar)

Kalikan kedua ruas dengan -1:
-2 ... -5   mana yang lebih besar?

Di garis bilangan, -2 ada di kanan -5.
Jadi: -2 > -5

Tanda berubah dari < menjadi >.
Kalau tidak dibalik: -2 < -5 (salah!).

Inilah kenapa aturan ini ada. Ia menjaga kebenaran pertidaksamaan setelah operasi perkalian/pembagian dengan bilangan negatif.

Prinsip Tambahan: Jangan Kali Silang Sembarangan

Di persamaan, kita bebas kali silang. Di pertidaksamaan, hati-hati! Kali silang sama dengan mengali dengan penyebut. Kalau penyebutnya bisa bernilai negatif, tanda bisa salah.

Aturan aman: Jangan kali silang di pertidaksamaan. Pindahkan semua ke satu ruas, samakan penyebut, lalu gunakan garis bilangan.

2. Pertidaksamaan Linear

Bentuk umum: ax + b < c (atau >, ≤, ≥), dengan a ≠ 0.

Caranya persis seperti persamaan linear, tapi perhatikan tanda saat membagi dengan bilangan negatif.

Contoh 1: Sederhana

Selesaikan: 3x + 5 > 14

3x > 14 - 5
3x > 9
x > 3

Solusi: x > 3 atau dalam interval: (3, ∞) ✅

Contoh 2: Tanda Dibalik

Selesaikan: -2x + 7 ≤ 1

-2x ≤ 1 - 7
-2x ≤ -6

Bagi dengan -2 (negatif!) → tanda dibalik:
x ≥ 3

Solusi: x ≥ 3 atau dalam interval: [3, ∞) ✅

Verifikasi: Coba x = 4: -2(4)+7 = -8+7 = -1 ≤ 1. Benar!
Coba x = 2: -2(2)+7 = -4+7 = 3. 3 ≤ 1? Salah. Jadi memang x ≥ 3.

Contoh 3: Variabel di Kedua Ruas

Selesaikan: 5x - 2 < 3x + 8

5x - 3x < 8 + 2
2x < 10
x < 5

Solusi: x < 5 atau (-∞, 5) ✅

3. Pertidaksamaan Kuadrat

Bentuk umum: ax² + bx + c < 0 (atau >, ≤, ≥), dengan a ≠ 0.

Berbeda dengan persamaan kuadrat yang solusinya berupa titik, pertidaksamaan kuadrat solusinya berupa interval.

Metode Garis Bilangan + Uji Titik

  1. Pindahkan semua ke satu ruas (ruas kanan = 0).
  2. Cari titik kritis: nilai x yang membuat ekspresi = 0 (akar-akar persamaan kuadratnya).
  3. Letakkan titik kritis pada garis bilangan. Garis terbagi menjadi beberapa interval.
  4. Uji satu titik dari setiap interval ke pertidaksamaan awal.
  5. Tentukan interval mana yang memenuhi.

Contoh 1

Selesaikan: x² - 5x + 6 < 0

Langkah 1: Cari titik kritis.
x² - 5x + 6 = 0
(x - 2)(x - 3) = 0
x = 2 atau x = 3

Langkah 2: Garis bilangan terbagi 3 interval:
Interval 1: x < 2
Interval 2: 2 < x < 3
Interval 3: x > 3

Langkah 3: Uji titik.
x = 0 (interval 1): 0 - 0 + 6 = 6. 6 < 0? Salah ❌
x = 2,5 (interval 2): 6,25 - 12,5 + 6 = -0,25. -0,25 < 0? Benar ✅
x = 4 (interval 3): 16 - 20 + 6 = 2. 2 < 0? Salah ❌

Solusi: 2 < x < 3 atau (2, 3) ✅

Mengapa uji titik bekerja? Fungsi kuadrat kontinu (tidak putus). Satu-satunya tempat di mana nilai fungsi bisa berubah tanda adalah di titik kritis (tempat fungsi = 0). Jadi, di dalam satu interval, semua titik punya tanda yang sama. Cukup uji satu titik per interval.

Contoh 2: Dengan Tanda ≥ (Titik Kritis Termasuk)

Selesaikan: x² - 4 ≥ 0

Titik kritis: x² - 4 = 0 → (x+2)(x-2) = 0 → x = -2, x = 2

Interval: x < -2, -2 < x < 2, x > 2

Uji:
x = -3: 9-4 = 5 ≥ 0 ✅
x = 0: 0-4 = -4 ≥ 0 ❌
x = 3: 9-4 = 5 ≥ 0 ✅

Titik kritis: x = -2 dan x = 2 → hasil = 0 ≥ 0 (memenuhi).

Solusi: x ≤ -2 atau x ≥ 2 atau (-∞, -2] ∪ [2, ∞) ✅

Contoh 3: a Negatif

Selesaikan: -x² + 3x + 4 > 0

Langkah 1: Kalikan dengan -1 (tanda dibalik):
x² - 3x - 4 < 0

Titik kritis: (x-4)(x+1) = 0 → x = 4, x = -1

Interval: x < -1, -1 < x < 4, x > 4

Uji:
x = -2: 4+6-4=6 < 0? ❌
x = 0: 0-0-4=-4 < 0? ✅
x = 5: 25-15-4=6 < 0? ❌

Solusi: -1 < x < 4 atau (-1, 4) ✅
💡 Trik Cepat: Untuk ax² + bx + c dengan a > 0 (parabola terbuka ke atas):
ax² + bx + c < 0 → solusinya di antara akar-akar.
ax² + bx + c > 0 → solusinya di luar akar-akar.

Untuk a < 0, kalikan dulu dengan -1 (balik tanda), baru pakai trik di atas.

4. Pertidaksamaan Rasional — Variabel di Penyebut

Bentuk: P(x)/Q(x) < 0 (atau >, ≤, ≥), di mana P dan Q adalah polinomial.

Strategi:

  1. Pindahkan semua ke satu ruas (ruas kanan = 0).
  2. Samakan penyebut.
  3. Cari titik kritis: nilai x yang membuat pembilang = 0 (titik nol) dan penyebut = 0 (titik singular/ tidak terdefinisi).
  4. Letakkan di garis bilangan. Uji tiap interval.
  5. Penting: Titik dari penyebut = 0 tidak pernah termasuk (karena tidak terdefinisi).

Contoh 1

Selesaikan: (x+1)/(x-2) > 0

Titik kritis:
Pembilang = 0: x + 1 = 0 → x = -1
Penyebut = 0: x - 2 = 0 → x = 2 (tidak termasuk!)

Interval: x < -1, -1 < x < 2, x > 2

Uji:
x = -2: (-1)/(-4) = 0,25 > 0 ✅
x = 0: 1/(-2) = -0,5 > 0 ❌
x = 3: 4/1 = 4 > 0 ✅

x = -1: 0/(-3) = 0. 0 > 0? Salah (karena >, bukan ≥). Jadi -1 tidak termasuk.
x = 2: tidak terdefinisi, otomatis tidak termasuk.

Solusi: x < -1 atau x > 2 atau (-∞, -1) ∪ (2, ∞) ✅

Contoh 2: Dengan Tanda ≤

Selesaikan: (x-3)/(x+1) ≤ 0

Titik kritis:
Pembilang = 0: x = 3 (termasuk karena ≤ memperbolehkan = 0)
Penyebut = 0: x = -1 (tidak termasuk)

Interval: x < -1, -1 < x < 3, x > 3

Uji:
x = -2: (-5)/(-1) = 5 ≤ 0 ❌
x = 0: (-3)/1 = -3 ≤ 0 ✅
x = 4: 1/5 = 0,2 ≤ 0 ❌

Solusi: -1 < x ≤ 3 atau (-1, 3] ✅

Contoh 3: Perlu Disamakan Penyebut Dulu

Selesaikan: 1/(x-1) ≥ 2/(x+2)

Langkah 1: Pindahkan ke satu ruas.
1/(x-1) - 2/(x+2) ≥ 0

Langkah 2: Samakan penyebut.
(x+2)/[(x-1)(x+2)] - 2(x-1)/[(x-1)(x+2)] ≥ 0
[x+2 - 2x + 2] / [(x-1)(x+2)] ≥ 0
(-x + 4) / [(x-1)(x+2)] ≥ 0
(4 - x) / [(x-1)(x+2)] ≥ 0

Titik kritis:
Pembilang = 0: x = 4
Penyebut = 0: x = 1, x = -2

Interval: x < -2, -2 < x < 1, 1 < x < 4, x > 4

Uji:
x = -3: (7)/((-4)(-1)) = 7/4 > 0 ✅
x = 0: 4/((-1)(2)) = -2 < 0 ❌
x = 2: 2/((1)(4)) = 0,5 > 0 ✅
x = 5: (-1)/((4)(7)) = -1/28 < 0 ❌

Titik x = 4: pembilang = 0 → ekspresi = 0 ≥ 0 (memenuhi, termasuk).

Solusi: x < -2 atau 1 < x ≤ 4 atau (-∞, -2) ∪ (1, 4] ✅

5. Pertidaksamaan Nilai Mutlak

Ingat definisi nilai mutlak: |x| adalah jarak x dari nol.

|x| < a (a > 0) → -a < x < a   (x berada dalam interval [-a, a])

|x| > a (a > 0) → x < -a atau x > a   (x berada di luar interval [-a, a])

Mengapa?

|x| < 5 berarti jarak x dari 0 kurang dari 5. Semua bilangan dari -5 sampai 5 (tidak termasuk ujung) memenuhi itu. |x| > 5 berarti jarak x dari 0 lebih dari 5. Semua bilangan di kiri -5 atau di kanan 5 memenuhi itu.

Contoh 1: Sederhana

Selesaikan: |x - 3| < 5

Ini berarti jarak x dari 3 kurang dari 5.

-5 < x - 3 < 5
-5 + 3 < x < 5 + 3
-2 < x < 8

Solusi: (-2, 8) ✅

Contoh 2: Lebih Besar

Selesaikan: |2x + 1| ≥ 7

2x + 1 ≤ -7   atau   2x + 1 ≥ 7

Untuk 2x + 1 ≤ -7:
2x ≤ -8 → x ≤ -4

Untuk 2x + 1 ≥ 7:
2x ≥ 6 → x ≥ 3

Solusi: x ≤ -4 atau x ≥ 3 atau (-∞, -4] ∪ [3, ∞) ✅

Contoh 3: Dengan Ekspresi Kuadrat

Selesaikan: |x² - 4| < 3

-3 < x² - 4 < 3

Kiri: -3 < x² - 4 → x² > 1 → x < -1 atau x > 1

Kanan: x² - 4 < 3 → x² < 7 → -√7 < x < √7

Irisan kedua solusi:
(-√7 < x < -1) ∪ (1 < x < √7)

Solusi: (-√7, -1) ∪ (1, √7) ✅

6. Pertidaksamaan Eksponen — Variabel di Pangkat

Pertidaksamaan eksponen adalah pertidaksamaan yang variabelnya muncul sebagai pangkat (eksponen).

2ˣ > 8
3²ˣ⁺¹ ≤ 27
5ˣ < 7ˣ⁻¹

Strategi kunci: samakan basis, lalu perhatikan apakah basis > 1 atau 0 < basis < 1. Ini menentukan apakah tanda tetap atau dibalik.

Aturan Penting: Basis Menentukan Arah Tanda

Jika basis > 1: aᵐ > aⁿ → m > n   (tanda tetap, fungsi naik)

Jika 0 < basis < 1: aᵐ > aⁿ → m < n   (tanda dibalik, fungsi turun)

Mengapa?

Untuk basis > 1 (misal 2): semakin besar pangkat, semakin besar nilainya. 2¹ = 2, 2² = 4, 2³ = 8. Nilai naik seiring pangkat naik.

Untuk basis antara 0 dan 1 (misal 1/2): semakin besar pangkat, semakin kecil nilainya. (1/2)¹ = 0,5, (1/2)² = 0,25, (1/2)³ = 0,125. Nilai turun seiring pangkat naik. Inilah kenapa tandanya harus dibalik.

Kasus 1: Basis > 1 (Tanda Tetap)

Selesaikan: 2ˣ > 8

Samakan basis: 8 = 2³
2ˣ > 2³
Basis 2 > 1 → tanda tetap: x > 3

Solusi: x > 3 atau (3, ∞) ✅
Selesaikan: 3²ˣ⁺¹ ≤ 27

27 = 3³
3²ˣ⁺¹ ≤ 3³
Basis 3 > 1 → tanda tetap:
2x + 1 ≤ 3
2x ≤ 2
x ≤ 1

Solusi: x ≤ 1 atau (-∞, 1] ✅

Kasus 2: 0 < Basis < 1 (Tanda Dibalik)

Selesaikan: (1/2)ˣ < 1/8

1/8 = (1/2)³
(1/2)ˣ < (1/2)³
Basis 1/2 (antara 0 dan 1) → tanda dibalik:
x > 3

Verifikasi:
x = 4: (1/2)⁴ = 1/16. 1/16 < 1/8? Ya (1/16 = 0,0625, 1/8 = 0,125). ✅
x = 2: (1/2)² = 1/4. 1/4 < 1/8? Tidak (0,25 > 0,125). ❌

Solusi: x > 3 atau (3, ∞) ✅

Kasus 3: Substitusi (Bentuk Kuadrat dalam Eksponen)

Selesaikan: 2²ˣ - 6·2ˣ + 8 < 0

Misalkan p = 2ˣ (p > 0 karena 2ˣ selalu positif):

p² - 6p + 8 < 0
(p - 2)(p - 4) < 0

Untuk p² - 6p + 8 < 0 (a > 0, parabola terbuka atas):
2 < p < 4

Kembalikan ke 2ˣ:
2 < 2ˣ < 4
2¹ < 2ˣ < 2²

Basis 2 > 1 → tanda tetap:
1 < x < 2

Solusi: 1 < x < 2 atau (1, 2) ✅

Kasus 4: Basis Berbeda

Selesaikan: 5ˣ < 7ˣ⁻¹

Basis berbeda, tidak bisa disamakan. Gunakan logaritma.

Ambil ln kedua ruas (ln adalah fungsi naik, tanda tetap):
ln(5ˣ) < ln(7ˣ⁻¹)
x·ln 5 < (x-1)·ln 7
x·ln 5 < x·ln 7 - ln 7
x·ln 5 - x·ln 7 < -ln 7
x(ln 5 - ln 7) < -ln 7

Karena ln 5 - ln 7 < 0 (negatif), bagi dengan bilangan negatif → tanda dibalik:
x > -ln 7 / (ln 5 - ln 7)
x > ln 7 / (ln 7 - ln 5)
x > ln 7 / ln(7/5)

Solusi: x > ln 7 / ln(7/5) (sekitar x > 5,78)

7. Pertidaksamaan Logaritma — Variabel di Dalam Log

Pertidaksamaan logaritma mengandung variabel di dalam operasi logaritma.

Strategi:

  1. Tentukan syarat numerus (numerus > 0).
  2. Samakan bentuk logaritma di kedua ruas (jika memungkinkan).
  3. Perhatikan basis: basis > 1 → tanda tetap; 0 < basis < 1 → tanda dibalik.
  4. Selesaikan pertidaksamaannya.
  5. Iriskan dengan syarat numerus.

Aturan Penting: Basis Menentukan Arah Tanda

Jika basis > 1: ᵃlog f(x) > ᵃlog g(x) → f(x) > g(x)   (tanda tetap)

Jika 0 < basis < 1: ᵃlog f(x) > ᵃlog g(x) → f(x) < g(x)   (tanda dibalik)

Mengapa? Karena fungsi logaritma dengan basis > 1 adalah fungsi naik (semakin besar numerus, semakin besar hasil log). Sedangkan fungsi logaritma dengan basis antara 0 dan 1 adalah fungsi turun (semakin besar numerus, semakin kecil hasil log).

Contoh 1: Basis > 1

Selesaikan: ²log(x+1) < 3

Syarat numerus: x + 1 > 0 → x > -1

Ubah 3 ke bentuk logaritma basis 2:
3 = ²log 8 (karena 2³ = 8)

²log(x+1) < ²log 8
Basis 2 > 1 → tanda tetap:
x + 1 < 8
x < 7

Iriskan dengan syarat: x > -1 dan x < 7

Solusi: -1 < x < 7 atau (-1, 7) ✅

Contoh 2: Basis Antara 0 dan 1 (Tanda Dibalik)

Selesaikan: ½log(x+2) > ½log(2x-1)

Syarat numerus:
x + 2 > 0 → x > -2
2x - 1 > 0 → x > 1/2
Gabungan: x > 1/2

Basis 1/2 (antara 0 dan 1) → tanda dibalik:
x + 2 < 2x - 1
2 + 1 < 2x - x
3 < x
x > 3

Iriskan dengan syarat: x > 3 dan x > 1/2 → x > 3

Solusi: x > 3 atau (3, ∞) ✅

Contoh 3: Bentuk Gabungan

Selesaikan: ³log(x² - 4) ≤ ³log 5

Syarat numerus: x² - 4 > 0 → (x+2)(x-2) > 0
→ x < -2 atau x > 2

Basis 3 > 1 → tanda tetap:
x² - 4 ≤ 5
x² ≤ 9
-3 ≤ x ≤ 3

Iriskan dengan syarat:
(-3 ≤ x ≤ 3) ∩ (x < -2 atau x > 2)

= [-3, -2) ∪ (2, 3]

Solusi: -3 ≤ x < -2 atau 2 < x ≤ 3 ✅
💡 Tips: Untuk pertidaksamaan logaritma, jangan pernah lupa syarat numerus > 0. Banyak solusi yang kelihatannya benar tapi melanggar syarat ini. Selalu tulis syarat di awal, dan iriskan di akhir.

8. Sistem Pertidaksamaan — Mencari Irisan

Sistem pertidaksamaan adalah kumpulan dua atau lebih pertidaksamaan yang harus dipenuhi secara bersamaan. Solusinya adalah irisan dari semua solusi masing-masing pertidaksamaan.

Strategi Umum

  1. Selesaikan setiap pertidaksamaan secara terpisah.
  2. Gambarkan solusi masing-masing pada garis bilangan yang sama.
  3. Cari irisan (daerah yang terkena semua arsiran).

Contoh 1: Sistem Linear

Selesaikan sistem:
2x + 1 > 5 ...(1)
x - 3 ≤ 2 ...(2)

Dari (1): 2x > 4 → x > 2

Dari (2): x ≤ 5

Irisan: x > 2 dan x ≤ 5

Solusi: 2 < x ≤ 5 atau (2, 5] ✅

Contoh 2: Sistem Linear-Kuadrat

Selesaikan sistem:
x² - 4 < 0 ...(1)
x + 1 > 0 ...(2)

Dari (1): (x+2)(x-2) < 0 → -2 < x < 2

Dari (2): x > -1

Irisan: (-2 < x < 2) ∩ (x > -1)

= -1 < x < 2 atau (-1, 2) ✅

Contoh 3: Sistem dengan Rasional

Selesaikan sistem:
(x-1)/(x+2) < 0 ...(1)
x ≥ 0 ...(2)

Dari (1):
Titik kritis: x = 1 (pembilang), x = -2 (penyebut, tidak termasuk).
Interval: x < -2, -2 < x < 1, x > 1
Uji: x=-3 → (+)/(-)=(-) < 0 ✅. x=0 → (-)/(+)=(-) < 0 ✅. x=2 → (+)/(+)=(+) < 0 ❌.
Solusi (1): x < -2 atau -2 < x < 1

Dari (2): x ≥ 0

Irisan: [(x < -2) ∪ (-2 < x < 1)] ∩ [x ≥ 0]
= 0 ≤ x < 1 atau [0, 1) ✅

Contoh 4: Tiga Pertidaksamaan

Selesaikan sistem:
x + 2 > 0 ...(1)
x² - 9 ≤ 0 ...(2)
x ≠ 1 ...(3)

Dari (1): x > -2

Dari (2): (x+3)(x-3) ≤ 0 → -3 ≤ x ≤ 3

Dari (3): x ≠ 1

Irisan (1) dan (2): (-2, ∞) ∩ [-3, 3] = (-2, 3]

Kemudian dengan (3): (-2, 3] kecuali x = 1

Solusi: -2 < x < 1 atau 1 < x ≤ 3 atau (-2, 1) ∪ (1, 3] ✅

📝 Ringkasan Cepat

  • Prinsip dasar: Kali/bagi dengan bilangan negatif → balik tanda.
  • Linear: Isolasi variabel, perhatikan tanda negatif.
  • Kuadrat: Cari titik kritis → garis bilangan → uji interval.
  • Rasional: Titik kritis dari pembilang dan penyebut. Penyebut = 0 → tidak termasuk.
  • Nilai mutlak: |x| < a → -a < x < a. |x| > a → x < -a atau x > a.
  • Eksponen: Basis > 1 → tanda tetap. 0 < basis < 1 → tanda dibalik.
  • Logaritma: Basis > 1 → tanda tetap. 0 < basis < 1 → tanda dibalik. Jangan lupa syarat numerus > 0.
  • Sistem: Selesaikan masing-masing → cari irisan.

✨ Tips & Trik

  • Garis bilangan adalah sahabatmu. Untuk pertidaksamaan kuadrat, rasional, dan sistem, menggambar garis bilangan membuat solusi terlihat jelas dan mengurangi kesalahan.
  • Uji titik itu wajib, bukan opsional. Jangan langsung menebak interval. Selalu uji satu titik dari setiap interval untuk memastikan tanda.
  • Hati-hati dengan titik dari penyebut. Di pertidaksamaan rasional, titik yang membuat penyebut = 0 tidak pernah termasuk solusi, meskipun tandanya ≤ atau ≥.
  • Jangan kali silang sembarangan. Di pertidaksamaan, kali silang berbahaya karena penyebut bisa negatif dan membalik tanda. Selalu pindahkan semua ke satu ruas dan samakan penyebut.
  • Untuk eksponen dan logaritma, cek basis dulu. Sebelum menyelesaikan, tentukan apakah basis > 1 atau 0 < basis < 1. Ini menentukan apakah tanda dibalik atau tidak.
  • Logaritma: syarat numerus adalah yang pertama. Tulis syarat numerus > 0 di awal, jangan di akhir. Banyak yang lupa dan dapat solusi yang salah.
  • Untuk sistem, gambar semua solusi di satu garis bilangan. Ini membantu visualisasi irisan dan mencegah kesalahan.
