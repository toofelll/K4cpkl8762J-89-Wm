Part 3 • Persamaan Lanjut & Sistem

📖 Pendahuluan

Di Bab 2, kita sudah menyelesaikan persamaan linear, kuadrat, dan kubik. Itu semua adalah persamaan polinomial—variabelnya muncul sebagai pangkat bulat.

Sekarang kita masuk ke jenis-jenis persamaan yang lebih "liar":

  • Persamaan rasional — variabel muncul di penyebut pecahan.
  • Persamaan irasional — variabel muncul di dalam tanda akar.
  • Persamaan eksponen — variabel muncul sebagai pangkat (eksponen).
  • Persamaan logaritma — variabel muncul di dalam logaritma.

Setelah itu, kita akan bahas sistem persamaan secara lebih luas: tidak hanya linear-linear, tapi juga campuran seperti linear-kuadrat dan kuadrat-kuadrat.

Setiap jenis persamaan punya "jebakan" masing-masing. Tapi tenang, kita akan bahas semuanya dengan contoh dan strategi yang jelas.

Tujuan Bab
  • Mampu menyelesaikan persamaan rasional dengan memperhatikan domain.
  • Mampu menyelesaikan persamaan irasional dan mengecek solusi palsu.
  • Memahami sifat-sifat eksponen untuk menyelesaikan persamaan eksponen.
  • Memahami sifat-sifat logaritma untuk menyelesaikan persamaan logaritma.
  • Mampu menyelesaikan sistem persamaan campuran (linear-kuadrat, kuadrat-kuadrat).

1. Persamaan Rasional — Variabel di Penyebut

Persamaan rasional adalah persamaan yang mengandung pecahan dengan variabel di penyebut.

3/(x+1) = 2
(x+2)/(x-3) = 4
1/x + 1/(x+1) = 5/6

Masalah utama: penyebut tidak boleh nol. Jadi, sebelum menyelesaikan, kita harus menentukan domain (syarat x).

Strategi Umum

  1. Tentukan syarat: penyebut ≠ 0.
  2. Kalikan kedua ruas dengan KPK penyebut untuk menghilangkan pecahan.
  3. Selesaikan persamaan yang dihasilkan (biasanya jadi linear atau kuadrat).
  4. Cek solusi ke syarat domain. Buang yang melanggar.

Contoh 1

Selesaikan: 3/(x+1) = 2

Syarat: x + 1 ≠ 0 → x ≠ -1

Kalikan kedua ruas dengan (x+1):
3 = 2(x+1)
3 = 2x + 2
1 = 2x
x = 1/2

Cek: x = 1/2 ≠ -1 (memenuhi syarat).

Verifikasi: 3/(1/2+1) = 3/(3/2) = 2. Cocok!

Solusi: x = 1/2 ✅

Contoh 2: Dua Pecahan

Selesaikan: 1/x + 1/(x+1) = 5/6

Syarat: x ≠ 0 dan x ≠ -1

KPK penyebut: 6x(x+1). Kalikan semua suku:
6x(x+1)·(1/x) + 6x(x+1)·(1/(x+1)) = 6x(x+1)·(5/6)
6(x+1) + 6x = 5x(x+1)
6x + 6 + 6x = 5x² + 5x
12x + 6 = 5x² + 5x
0 = 5x² - 7x - 6
5x² - 7x - 6 = 0

Faktorkan:
(5x + 3)(x - 2) = 0
x = -3/5 atau x = 2

Cek syarat: x = -3/5 ≠ 0, -1 ✅. x = 2 ≠ 0, -1 ✅.

Verifikasi:
Untuk x = 2: 1/2 + 1/3 = 3/6 + 2/6 = 5/6 ✅
Untuk x = -3/5: -5/3 + 5/2 = -10/6 + 15/6 = 5/6 ✅

Solusi: x = 2 atau x = -3/5

Contoh 3: Solusi yang Ditolak

Selesaikan: (x+2)/(x-3) = (x-1)/(x-3)

Syarat: x ≠ 3

Karena penyebutnya sama, kita bisa langsung menyamakan pembilang:
x + 2 = x - 1
2 = -1 (kontradiksi!)

Tidak ada solusi. Persamaan ini tidak memiliki solusi.

Kenapa? Karena satu-satunya kemungkinan adalah x = 3 (penyebut 0), tapi itu justru dilarang.

2. Persamaan Irasional — Variabel di Dalam Akar

Persamaan irasional adalah persamaan yang mengandung variabel di dalam tanda akar.

√(x+3) = 5
√(2x-1) = x - 2
√(x+5) + √(x-3) = 4

Masalah utama:

  • Radikan (isi akar) tidak boleh negatif untuk akar genap → syarat domain.
  • Pengkuadratan bisa menghasilkan solusi palsu! Ini jebakan paling umum.

Mengapa Pengkuadratan Bisa Menghasilkan Solusi Palsu?

Saat kita mengkuadratkan kedua ruas, kita kehilangan informasi tanda.

Misal: x = 5
Kuadratkan: x² = 25
Solusi: x = 5 atau x = -5

Padahal persamaan aslinya hanya x = 5. x = -5 adalah solusi palsu yang muncul karena pengkuadratan.

Intinya: Jika A = B, maka A² = B² (pernyataan benar). Tapi jika A² = B², belum tentu A = B (bisa jadi A = -B).

Karena itu, setiap kali mengkuadratkan dalam persamaan irasional, selalu substitusi balik solusi ke persamaan awal untuk verifikasi.

Contoh 1: Satu Akar

Selesaikan: √(x+3) = 5

Syarat: x + 3 ≥ 0 → x ≥ -3

Kuadratkan kedua ruas:
(√(x+3))² = 5²
x + 3 = 25
x = 22

Cek syarat: 22 ≥ -3 ✅
Verifikasi: √(22+3) = √25 = 5 ✅

Solusi: x = 22

Contoh 2: Menghasilkan Solusi Palsu

Selesaikan: √(2x-1) = x - 2

Syarat:
Radikan: 2x - 1 ≥ 0 → x ≥ 1/2
Ruas kanan tidak negatif (karena hasil akar ≥ 0): x - 2 ≥ 0 → x ≥ 2
Gabungan: x ≥ 2

Kuadratkan:
2x - 1 = (x - 2)²
2x - 1 = x² - 4x + 4
0 = x² - 6x + 5
0 = (x - 1)(x - 5)
x = 1 atau x = 5

Cek syarat: x = 1 tidak memenuhi x ≥ 2 ❌
x = 5 memenuhi x ≥ 2 ✅

Verifikasi x = 5: √(10-1) = √9 = 3. 5 - 2 = 3. Cocok! ✅

Solusi: x = 5 (x = 1 adalah solusi palsu yang muncul dari pengkuadratan)

Perhatikan: x = 1 muncul dari faktorisasi, tapi tidak memenuhi syarat awal. Kalau kita tidak mengecek, kita akan mengira ada dua solusi.

Contoh 3: Dua Akar

Selesaikan: √(x+5) + √(x-3) = 4

Syarat:
x + 5 ≥ 0 → x ≥ -5
x - 3 ≥ 0 → x ≥ 3
Gabungan: x ≥ 3

Pisahkan salah satu akar:
√(x+5) = 4 - √(x-3)

Kuadratkan:
x + 5 = 16 - 8√(x-3) + (x-3)
x + 5 = 16 + x - 3 - 8√(x-3)
x + 5 = x + 13 - 8√(x-3)
5 = 13 - 8√(x-3)
-8 = -8√(x-3)
1 = √(x-3)

Kuadratkan lagi:
1 = x - 3
x = 4

Cek syarat: 4 ≥ 3 ✅
Verifikasi: √(4+5) + √(4-3) = √9 + √1 = 3 + 1 = 4 ✅

Solusi: x = 4

3. Persamaan Eksponen — Variabel di Pangkat

Persamaan eksponen adalah persamaan yang variabelnya muncul sebagai pangkat (eksponen).

2ˣ = 8
3²ˣ⁺¹ = 27
5ˣ = 7ˣ⁻¹
2²ˣ - 5·2ˣ + 4 = 0

Strategi kunci: gunakan sifat-sifat eksponen untuk menyamakan basis atau melakukan substitusi.

Sifat-Sifat Eksponen yang Dipakai

aᵐ × aⁿ = aᵐ⁺ⁿ
aᵐ ÷ aⁿ = aᵐ⁻ⁿ
(aᵐ)ⁿ = aᵐⁿ
a⁰ = 1 (a ≠ 0)
a⁻ⁿ = 1/aⁿ
Jika aᵐ = aⁿ dan a > 0, a ≠ 1, maka m = n

Sifat terakhir adalah kunci utama persamaan eksponen.

Kasus 1: Basis Sama → Samakan Pangkat

Selesaikan: 2ˣ = 8

Ubah ke basis yang sama: 8 = 2³
2ˣ = 2³ → x = 3 ✅


Selesaikan: 3²ˣ⁺¹ = 27

27 = 3³
3²ˣ⁺¹ = 3³
2x + 1 = 3
2x = 2
x = 1 ✅

Kasus 2: Basis Berbeda, Pangkat Sama → Selesaikan

Selesaikan: 5ˣ = 7ˣ⁻¹

Ini tidak bisa disamakan basisnya. Strategi: ubah bentuk.

5ˣ = 7ˣ / 7
7·5ˣ = 7ˣ
7 = 7ˣ / 5ˣ
7 = (7/5)ˣ

Sekarang basisnya (7/5):
(7/5)ˣ = 7
x = log_(7/5)(7)

Atau dengan logaritma natural:
x ln(7/5) = ln 7
x = ln 7 / ln(7/5)

Ini sudah solusi eksaknya. Untuk nilai numerik: x ≈ 5,78.

Kasus 3: Substitusi (Bentuk Kuadrat dalam Eksponen)

Selesaikan: 2²ˣ - 5·2ˣ + 4 = 0

Misalkan p = 2ˣ, maka 2²ˣ = (2ˣ)² = p².

p² - 5p + 4 = 0
(p - 1)(p - 4) = 0
p = 1 atau p = 4

Kembalikan ke 2ˣ:
2ˣ = 1 → 2ˣ = 2⁰ → x = 0
2ˣ = 4 → 2ˣ = 2² → x = 2

Solusi: x = 0 atau x = 2 ✅

Verifikasi:
x = 0: 2⁰ - 5·2⁰ + 4 = 1 - 5 + 4 = 0 ✅
x = 2: 2⁴ - 5·2² + 4 = 16 - 20 + 4 = 0 ✅

Kasus 4: Basis Berbeda, Pangkat Berbeda → Logaritma

Selesaikan: 3ˣ = 5ˣ⁺²

Tidak bisa disamakan basis. Ambil logaritma kedua ruas:
ln(3ˣ) = ln(5ˣ⁺²)
x·ln 3 = (x+2)·ln 5
x·ln 3 = x·ln 5 + 2·ln 5
x·ln 3 - x·ln 5 = 2·ln 5
x(ln 3 - ln 5) = 2·ln 5
x = 2·ln 5 / (ln 3 - ln 5)

Ini adalah solusi eksaknya. Bisa ditulis juga sebagai x = 2 / (log₃5 - 1).

4. Persamaan Logaritma — Variabel di Dalam Log

Logaritma adalah operasi kebalikan dari eksponen.

Jika aᶜ = b, maka ᵃlog b = c

a disebut basis, b disebut numerus, c adalah hasil logaritma.

Syarat: a > 0, a ≠ 1, dan b > 0.

Persamaan logaritma mengandung variabel di dalam operasi logaritma.

Sifat-Sifat Logaritma yang Penting

1. ᵃlog 1 = 0 (karena a⁰ = 1)
2. ᵃlog a = 1 (karena a¹ = a)
3. ᵃlog(b·c) = ᵃlog b + ᵃlog c
4. ᵃlog(b/c) = ᵃlog b - ᵃlog c
5. ᵃlog bⁿ = n·ᵃlog b
6. ᵃlog b = (ᶜlog b) / (ᶜlog a)   (perubahan basis)
7. a^(ᵃlog b) = b   (invers)
8. Jika ᵃlog f(x) = ᵃlog g(x), maka f(x) = g(x) (dengan syarat f(x), g(x) > 0)

Kasus 1: Log Tunggal → Definisi

Selesaikan: ²log(x+1) = 3

Gunakan definisi: ²log(x+1) = 3 artinya 2³ = x+1
8 = x + 1
x = 7

Syarat: x + 1 > 0 → 8 > 0 ✅

Solusi: x = 7 ✅

Kasus 2: Log Sama → Numerus Sama

Selesaikan: ³log(2x-1) = ³log(x+3)

Karena basis sama: 2x - 1 = x + 3
x = 4

Syarat numerus > 0:
2(4)-1 = 7 > 0 ✅
4+3 = 7 > 0 ✅

Solusi: x = 4 ✅

Kasus 3: Gabungan Sifat Logaritma

Selesaikan: ²log(x+2) + ²log(x-2) = 3

Gunakan sifat 3: ²log((x+2)(x-2)) = 3
²log(x² - 4) = 3

Definisi: 2³ = x² - 4
8 = x² - 4
x² = 12
x = ±√12 = ±2√3

Syarat numerus:
Untuk x = 2√3 ≈ 3,46: x+2 > 0 dan x-2 > 0 ✅
Untuk x = -2√3 ≈ -3,46: x+2 < 0 ❌

Solusi: x = 2√3 ✅

Kasus 4: Substitusi

Selesaikan: (²log x)² - 3·²log x + 2 = 0

Misalkan p = ²log x:
p² - 3p + 2 = 0
(p - 1)(p - 2) = 0
p = 1 atau p = 2

Kembalikan:
²log x = 1 → x = 2¹ = 2
²log x = 2 → x = 2² = 4

Syarat: x > 0 → 2 > 0 ✅, 4 > 0 ✅

Solusi: x = 2 atau x = 4 ✅

5. Sistem Persamaan Campuran

Di Bab 2, kita sudah membahas sistem persamaan linear-linear. Sekarang kita bahas sistem campuran: linear-kuadrat dan kuadrat-kuadrat.

Sistem Linear-Kuadrat

Bentuk: Satu persamaan linear + satu persamaan kuadrat.

Strategi: Substitusi dari persamaan linear ke persamaan kuadrat.

Selesaikan sistem:
y = x + 1 ...(1)
x² + y² = 25 ...(2)

Substitusi (1) ke (2):
x² + (x+1)² = 25
x² + x² + 2x + 1 = 25
2x² + 2x - 24 = 0
x² + x - 12 = 0
(x + 4)(x - 3) = 0
x = -4 atau x = 3

Untuk x = -4: y = -3
Untuk x = 3: y = 4

Solusi: (-4, -3) dan (3, 4) ✅

Interpretasi geometris: Ini adalah perpotongan garis lurus dengan lingkaran. Bisa ada 0, 1, atau 2 titik potong.

Sistem Kuadrat-Kuadrat

Bentuk: Dua persamaan kuadrat.

Strategi: Eliminasi salah satu variabel (biasanya dengan pengurangan), lalu selesaikan.

Selesaikan sistem:
x² + y² = 25 ...(1)
x² - y² = 7 ...(2)

Jumlahkan (1) dan (2):
2x² = 32
x² = 16
x = ±4

Kurangkan (1) - (2):
2y² = 18
y² = 9
y = ±3

Kombinasikan semua kemungkinan:
(4, 3), (4, -3), (-4, 3), (-4, -3)

Verifikasi: Semua memenuhi x²+y²=25 dan x²-y²=7 ✅

Solusi: 4 pasangan

Sistem Tiga Variabel dengan Campuran

Selesaikan sistem:
x + y = 3 ...(1)
y + z = 5 ...(2)
x² + z² = 10 ...(3)

Dari (1): y = 3 - x
Dari (2): z = 5 - y = 5 - (3-x) = 2 + x

Substitusi ke (3):
x² + (x+2)² = 10
x² + x² + 4x + 4 = 10
2x² + 4x - 6 = 0
x² + 2x - 3 = 0
(x + 3)(x - 1) = 0
x = -3 atau x = 1

Untuk x = -3: y = 6, z = -1
Untuk x = 1: y = 2, z = 3

Verifikasi semua:
(-3,6,-1): x+y=3 ✅, y+z=5 ✅, x²+z²=9+1=10 ✅
(1,2,3): x+y=3 ✅, y+z=5 ✅, x²+z²=1+9=10 ✅

Solusi: (-3, 6, -1) dan (1, 2, 3)

📝 Ringkasan Cepat

  • Rasional: Kalikan KPK penyebut, cek domain (penyebut ≠ 0).
  • Irasional: Kuadratkan, tapi wajib verifikasi karena bisa muncul solusi palsu.
  • Eksponen: Samakan basis, atau substitusi, atau gunakan logaritma.
  • Logaritma: Gunakan sifat log, samakan numerus, selalu cek syarat (numerus > 0).
  • Sistem campuran: Substitusi atau eliminasi, hasilnya bisa lebih dari satu solusi.

✨ Tips & Trik

  • Persamaan rasional: Jangan lupa tulis syarat domain di awal. Ini menyelamatkanmu dari solusi yang tidak valid.
  • Persamaan irasional: Setelah mengkuadratkan, selalu substitusi balik. Jangan skip langkah ini. Solusi palsu adalah jebakan klasik.
  • Persamaan eksponen: Kalau bingung, ubah semua ke bentuk aᵐ = aⁿ. Kalau basis tidak bisa disamakan, logaritma adalah penyelamat.
  • Persamaan logaritma: Selalu tulis syarat numerus > 0 dan basis > 0, ≠ 1. Banyak solusi ditolak karena melanggar syarat ini.
  • Sistem persamaan: Untuk sistem campuran, gambar sketsa (kalau bisa) untuk memperkirakan jumlah solusi. Lingkaran + garis bisa 0, 1, atau 2 titik potong.
  • Verifikasi adalah kebiasaan emas. Di semua jenis persamaan ini, verifikasi bisa membedakan jawaban benar dan solusi palsu.
