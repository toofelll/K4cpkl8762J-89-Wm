📊 KALKULUS

Part 5 • Integral: Konsep & Teknik Dasar

📖 Pendahuluan — Kebalikan dari Turunan

Sejauh ini kita telah mempelajari kalkulus diferensial—cabang kalkulus yang berfokus pada laju perubahan dan kemiringan kurva. Sekarang kita memasuki kalkulus integral—cabang yang berfokus pada akumulasi dan luas.

Jika turunan menjawab "seberapa cepat?", integral menjawab "berapa totalnya?". Jika turunan mencari kemiringan garis singgung, integral mencari luas daerah di bawah kurva. Keduanya adalah operasi yang saling berkebalikan— seperti penjumlahan dan pengurangan, perkalian dan pembagian.

Hubungan mendalam antara turunan dan integral diungkapkan oleh Teorema Dasar Kalkulus (Fundamental Theorem of Calculus)— salah satu pencapaian intelektual terbesar umat manusia. Teorema ini menyatakan bahwa integral dari turunan kembali ke fungsi semula, dan turunan dari integral juga kembali ke fungsi semula.

Secara historis, integral ditemukan lebih dulu daripada turunan. Archimedes (287-212 SM) sudah menggunakan metode "exhaustion" (penjumlahan tak hingga) untuk menghitung luas dan volume— jauh sebelum Newton dan Leibniz merumuskan kalkulus modern. Tapi hubungan antara integral dan turunan baru dipahami pada abad ke-17.

Tujuan Bab
  • Memahami integral tak tentu sebagai "anti-turunan" (kebalikan dari turunan).
  • Memahami integral tentu sebagai limit dari jumlah Riemann—luas di bawah kurva.
  • Menguasai Teorema Dasar Kalkulus yang menghubungkan integral tak tentu dan integral tentu.
  • Mampu menghitung integral dengan aturan pangkat, eksponen, trigonometri, dan substitusi.

1. Integral Tak Tentu — "Anti-Turunan"

Integral tak tentu (indefinite integral) adalah operasi kebalikan dari turunan.

Jika F'(x) = f(x), maka F adalah anti-turunan dari f, ditulis:

∫ f(x) dx = F(x) + C

C adalah konstanta integrasi—sangat penting dan tidak boleh dilupakan!

Mengapa ada "+ C"? — Konstanta yang Hilang

Turunan dari x² adalah 2x.
Turunan dari x² + 5 adalah 2x.
Turunan dari x² - 100 adalah 2x.
Turunan dari x² + π adalah 2x.

Semua fungsi ini memiliki turunan yang sama: 2x.

Jadi, ketika kita membalikkan prosesnya ("siapa yang turunannya 2x?"), ada tak hingga kemungkinan jawaban! Semua fungsi berbentuk x² + C memiliki turunan 2x.

Konstanta C mewakili semua kemungkinan itu. Inilah kenapa setiap integral tak tentu harus diakhiri dengan + C.

1.1 Aturan Pangkat — Kebalikan dari Turunan

Aturan Pangkat untuk Integral:

∫ xⁿ dx = xⁿ⁺¹/(n+1) + C,   untuk n ≠ -1

∫ x⁻¹ dx = ∫ (1/x) dx = ln |x| + C

Mengapa aturannya seperti itu? — Cek dengan Turunan

d/dx [xⁿ⁺¹/(n+1)] = (n+1)xⁿ/(n+1) = xⁿ. ✅

d/dx [ln |x|] = 1/x. ✅

Jadi aturan pangkat untuk integral adalah kebalikan persis dari aturan pangkat untuk turunan: turunan mengurangi pangkat 1, integral menambah pangkat 1 (lalu dibagi dengan pangkat baru).
∫ x³ dx = x⁴/4 + C. Cek: d/dx(x⁴/4) = 4x³/4 = x³. ✅

∫ x⁵ dx = x⁶/6 + C. Cek: d/dx(x⁶/6) = 6x⁵/6 = x⁵. ✅

∫ √x dx = ∫ x^(1/2) dx = x^(3/2)/(3/2) + C = (2/3)x^(3/2) + C. ✅

∫ 1/x² dx = ∫ x⁻² dx = x⁻¹/(-1) + C = -1/x + C. ✅

1.2 Sifat-Sifat Integral — Linieritas

1. ∫ k·f(x) dx = k·∫ f(x) dx (konstanta bisa dikeluarkan).

2. ∫ [f(x) ± g(x)] dx = ∫ f(x) dx ± ∫ g(x) dx (integral jumlah = jumlah integral).

3. ∫ dx = x + C (integral dari 1 adalah x).
∫ (3x² + 4x - 5) dx = 3∫x² dx + 4∫x dx - 5∫dx
= 3(x³/3) + 4(x²/2) - 5x + C
= x³ + 2x² - 5x + C. ✅

1.3 Integral Fungsi-Fungsi Penting — Tabel Wajib Hafal

∫ xⁿ dx = xⁿ⁺¹/(n+1) + C   (n ≠ -1)
∫ 1/x dx = ln |x| + C
∫ eˣ dx = eˣ + C
∫ aˣ dx = aˣ/ln a + C
∫ sin x dx = -cos x + C   (perhatikan tanda minus!)
∫ cos x dx = sin x + C
∫ sec² x dx = tan x + C
∫ csc² x dx = -cot x + C
∫ sec x tan x dx = sec x + C
∫ csc x cot x dx = -csc x + C

Mengapa ∫ sin x dx = -cos x? Kenapa ada tanda minus?

Karena d/dx(cos x) = -sin x, maka d/dx(-cos x) = sin x.

Jadi anti-turunan dari sin x adalah -cos x (bukan cos x!).

Ini adalah jebakan klasik: turunan cos x = -sin x, tapi turunan -cos x = sin x. Tanda minusnya "pindah".

2. Integral Tentu — Luas di Bawah Kurva

Integral tentu (definite integral) dari f(x) dari a ke b adalah luas bertanda daerah antara kurva y = f(x) dan sumbu X, dari x = a sampai x = b.

Notasi: ∫ₐᵇ f(x) dx

a = batas bawah, b = batas atas.

Hasilnya adalah bilangan (bukan fungsi), tidak ada + C!

Luas di atas sumbu X → positif. Luas di bawah sumbu X → negatif.

2.1 Dari Mana Integral Tentu Berasal? — Jumlah Riemann

Motivasi: Bagaimana menghitung luas daerah di bawah kurva yang melengkung? Kita tidak bisa menggunakan rumus persegi atau segitiga biasa.

Strategi: Potong daerah menjadi n persegi panjang tipis, hitung luas masing-masing, jumlahkan, lalu biarkan n → ∞.

Jumlah Riemann:

Bagilah [a, b] menjadi n subinterval dengan lebar Δx = (b-a)/n.
Pilih titik sampel xᵢ* di setiap subinterval.

Jumlah Riemann: Σ[i=1..n] f(xᵢ*) · Δx

Integral tentu = limit jumlah Riemann saat n → ∞:

∫ₐᵇ f(x) dx = lim[n→∞] Σ[i=1..n] f(xᵢ*) · Δx

Mengapa ini bekerja? Semakin banyak persegi panjang (n besar), semakin kecil Δx, dan semakin akurat aproksimasinya. Saat n → ∞, jumlah persegi panjang tepat sama dengan luas di bawah kurva.

Hitung ∫₀² x dx menggunakan limit jumlah Riemann.

Δx = (2-0)/n = 2/n. Titik sampel: ujung kanan xᵢ = 2i/n.

Jumlah Riemann = Σ f(2i/n)·(2/n) = Σ (2i/n)·(2/n) = (4/n²) Σ i.

Σ[i=1..n] i = n(n+1)/2.

Jumlah Riemann = (4/n²) · n(n+1)/2 = 2(n+1)/n = 2 + 2/n.

lim[n→∞] (2 + 2/n) = 2. ✅

Luas segitiga dengan alas 2 dan tinggi 2 = ½·2·2 = 2. Cocok!

2.2 Sifat-Sifat Integral Tentu

1. ∫ₐᵃ f(x) dx = 0 (batas sama → luas nol).

2. ∫ₐᵇ f(x) dx = -∫ᵇₐ f(x) dx (membalik batas → tanda berubah).

3. ∫ₐᵇ f(x) dx + ∫ᵇᶜ f(x) dx = ∫ₐᶜ f(x) dx (aditif terhadap interval).

4. ∫ₐᵇ k·f(x) dx = k·∫ₐᵇ f(x) dx (konstanta bisa dikeluarkan).

5. ∫ₐᵇ [f(x) ± g(x)] dx = ∫ₐᵇ f(x) dx ± ∫ₐᵇ g(x) dx.

3. Teorema Dasar Kalkulus — Jembatan Turunan dan Integral

Ini adalah teorema paling penting dalam kalkulus. Ia menghubungkan dua operasi yang tampak berbeda: turunan (laju perubahan) dan integral (akumulasi). Teorema ini memiliki dua bagian.

3.1 Bagian 1 — Turunan dari Integral

Teorema Dasar Kalkulus Bagian 1 (FTC1):

Jika f kontinu pada [a, b] dan F(x) = ∫ₐˣ f(t) dt, maka:

F'(x) = f(x)

Artinya: turunan dari integral (dengan batas atas variabel) kembali ke fungsi aslinya!

Makna: Jika kamu mengakumulasi f(t) dari a sampai x, lalu kamu mencari laju perubahan akumulasi itu terhadap x, kamu mendapatkan kembali f(x). Integral lalu turunan = fungsi semula.

F(x) = ∫₁ˣ t² dt. Maka F'(x) = x². (Tanpa perlu menghitung integralnya!)

Verifikasi: F(x) = [t³/3]₁ˣ = x³/3 - 1/3. F'(x) = 3x²/3 = x². ✅

3.2 Bagian 2 — Menghitung Integral Tentu

Teorema Dasar Kalkulus Bagian 2 (FTC2):

Jika f kontinu pada [a, b] dan F adalah anti-turunan dari f (F' = f), maka:

∫ₐᵇ f(x) dx = F(b) - F(a)

Artinya: integral tentu = anti-turunan di batas atas dikurangi anti-turunan di batas bawah!

Mengapa ini revolusioner? Sebelum FTC2, menghitung luas di bawah kurva harus menggunakan limit jumlah Riemann yang sangat rumit. Dengan FTC2, kita cukup mencari anti-turunan, lalu substitusi batas. Proses yang tadinya berhalaman-halaman kini menjadi beberapa baris!

Hitung ∫₀² x² dx.

Anti-turunan x² = x³/3. (Cek: d/dx(x³/3) = x²)

∫₀² x² dx = [x³/3]₀² = (8/3) - (0/3) = 8/3. ✅

Bandingkan dengan limit Riemann—jauh lebih cepat!
Hitung ∫₀ᵖⁱ sin x dx.

Anti-turunan sin x = -cos x.

∫₀ᵖⁱ sin x dx = [-cos x]₀ᵖⁱ = (-cos π) - (-cos 0)
= (-(-1)) - (-1) = 1 + 1 = 2. ✅

Luas total satu "gunung" gelombang sinus = tepat 2 satuan luas!

3.3 Notasi Evaluasi — Cara Penulisan yang Rapi

Notasi: F(x)|ₐᵇ atau [F(x)]ₐᵇ artinya F(b) - F(a).

Contoh: ∫₁³ 2x dx = x²|₁³ = 9 - 1 = 8.

4. Teknik Substitusi — "Aturan Rantai Terbalik"

Sama seperti aturan rantai adalah teknik paling penting dalam turunan, substitusi adalah teknik paling penting dalam integral. Substitusi adalah aturan rantai yang dibalik.

Rumus Substitusi:

∫ f(g(x)) · g'(x) dx = ∫ f(u) du

di mana u = g(x) dan du = g'(x) dx.

Strategi:
1. Pilih u = g(x) (biasanya "fungsi dalam").
2. Hitung du = g'(x) dx.
3. Ubah semua x menjadi u (termasuk dx).
4. Integralkan dalam u.
5. Kembalikan u ke x.

Untuk integral tentu: ubah juga batas integral ke u!

Mengapa substitusi adalah "aturan rantai terbalik"?

Aturan rantai: d/dx [F(g(x))] = F'(g(x)) · g'(x).

Ambil integral kedua ruas:
F(g(x)) = ∫ F'(g(x)) · g'(x) dx.

Misalkan f = F' (artinya F adalah anti-turunan f):
∫ f(g(x)) · g'(x) dx = F(g(x)) + C = ∫ f(u) du. ✅

4.1 Contoh Substitusi Dasar

Contoh 1: ∫ 2x·cos(x²) dx.

Pilih u = x² → du = 2x dx. Perhatikan: 2x dx sudah ada!

∫ cos(x²) · 2x dx = ∫ cos u du = sin u + C = sin(x²) + C. ✅

Contoh 2: ∫ (3x² + 1)⁴ · 6x dx.

u = 3x² + 1 → du = 6x dx. Pas!

= ∫ u⁴ du = u⁵/5 + C = (3x²+1)⁵/5 + C. ✅

4.2 Substitusi yang Perlu Penyesuaian

Contoh 3: ∫ x·√(x²+1) dx.

u = x² + 1 → du = 2x dx → x dx = du/2.

= ∫ √u · (du/2) = (1/2)∫ u^(1/2) du
= (1/2) · u^(3/2)/(3/2) + C = (1/2) · (2/3)u^(3/2) + C
= (1/3)(x²+1)^(3/2) + C. ✅

4.3 Substitusi pada Integral Tentu — Ubah Batas!

Contoh 4: ∫₀¹ 2x(x²+1)³ dx.

u = x² + 1. du = 2x dx.
Saat x = 0: u = 0²+1 = 1.
Saat x = 1: u = 1²+1 = 2.

= ∫₁² u³ du = [u⁴/4]₁² = 16/4 - 1/4 = 15/4. ✅

📝 Ringkasan Cepat Kalkulus 5

  • Integral tak tentu: ∫ f(x) dx = F(x) + C. Anti-turunan = kebalikan turunan. +C wajib!
  • Aturan pangkat: ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C (n ≠ -1). ∫ 1/x dx = ln |x| + C.
  • Integral tentu: ∫ₐᵇ f(x) dx = luas bertanda di bawah kurva dari a ke b.
  • Jumlah Riemann: Σ f(xᵢ*)Δx. Integral tentu = limit jumlah Riemann saat n→∞.
  • FTC1: d/dx ∫ₐˣ f(t) dt = f(x). Turunan dari integral kembali ke fungsi.
  • FTC2: ∫ₐᵇ f(x) dx = F(b) - F(a). Integral = anti-turunan di batas atas - bawah.
  • Substitusi: u = g(x), du = g'(x)dx. Aturan rantai terbalik. Ubah batas untuk integral tentu!

✨ Tips & Trik

  • +C jangan pernah dilupakan! Integral tak tentu selalu berakhir dengan +C. Tanpa +C, jawabanmu salah—meskipun perhitungannya benar.
  • Cek dengan turunan. Setelah mengintegralkan, turunkan hasilnya. Kalau kembali ke fungsi awal, integralmu benar. Ini verifikasi termudah.
  • Substitusi: pilih u = "fungsi dalam". Cari bagian yang kalau diturunkan, hasilnya muncul di integral. Biasanya yang di dalam kurung, di dalam akar, atau di dalam trigonometri.
  • Integral tentu substitusi: jangan lupa ubah batas! Kalau tidak mengubah batas, kamu harus mengembalikan u ke x dulu sebelum memasukkan batas x.
  • Integral = luas bertanda. Kalau kurva di bawah sumbu X, integralnya negatif. Luas total = |integral bagian atas| + |integral bagian bawah|.
  • Hafalkan integral dasar seperti turunan dasar. ∫sin = -cos, ∫cos = sin, ∫eˣ = eˣ, ∫1/x = ln|x|. Ini harus refleks.
