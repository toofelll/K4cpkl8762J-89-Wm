📊 KALKULUS

Part 1 • Fungsi & Limit

📖 Pendahuluan — Dua Pilar Kalkulus

Kalkulus adalah cabang matematika yang mempelajari perubahan. Ia dibangun di atas dua pilar raksasa: turunan (differential calculus) yang mempelajari laju perubahan sesaat, dan integral (integral calculus) yang mempelajari akumulasi kuantitas.

Tapi sebelum kita masuk ke turunan dan integral, kita harus memahami satu konsep yang menjadi fondasi seluruh kalkulus: limit. Limit menjawab pertanyaan: "Nilai apa yang didekati oleh suatu fungsi ketika inputnya mendekati suatu nilai tertentu?" Tanpa limit, turunan (yang melibatkan pembagian dengan selisih yang menuju nol) dan integral (yang melibatkan penjumlahan tak hingga) tidak bisa didefinisikan secara rigor.

Kalkulus modern dikembangkan secara independen oleh Isaac Newton di Inggris dan Gottfried Wilhelm Leibniz di Jerman pada akhir abad ke-17. Newton menggunakannya untuk menjelaskan gerak planet dan gravitasi. Leibniz mengembangkan notasi yang kita pakai sampai sekarang (dy/dx, ∫). Persaingan sengit antara keduanya adalah salah satu drama terbesar dalam sejarah sains.

Namun, definisi limit yang benar-benar rigor (epsilon-delta) baru dirumuskan oleh Augustin-Louis Cauchy dan Karl Weierstrass pada abad ke-19—hampir 200 tahun setelah Newton dan Leibniz.

Tujuan Bab
  • Memahami konsep fungsi sebagai mesin input-output dan berbagai jenisnya.
  • Memahami konsep limit secara intuitif: "nilai yang didekati".
  • Menguasai cara menghitung limit fungsi aljabar sederhana.
  • Memahami limit sepihak (kiri dan kanan).
  • Memahami limit tak hingga dan limit di tak hingga.
  • Memahami konsep kontinuitas dan Teorema Nilai Antara.

1. Fungsi — Fondasi yang Harus Kuat

Sebelum masuk ke limit, kita perlu memastikan pemahaman tentang fungsi benar-benar solid. Fungsi adalah jantung dari kalkulus. Setiap konsep dalam kalkulus—limit, turunan, integral—adalah operasi yang diterapkan pada fungsi.

Fungsi f adalah aturan yang memasangkan setiap input x (dari domain) dengan tepat satu output y = f(x) (di range).

Notasi: f : A → B, di mana A = domain, B = kodomain.

Domain: himpunan semua input yang valid (tidak menyebabkan pembagian nol, akar negatif, log non-positif).

Range: himpunan semua output aktual yang dihasilkan.

Mengapa "tepat satu"? Karena fungsi harus deterministik: input yang sama harus selalu menghasilkan output yang sama. Ini adalah sifat fundamental yang membedakan fungsi dari relasi biasa.

1.1 Jenis-Jenis Fungsi yang Penting di Kalkulus

1. Fungsi Polinomial: f(x) = aₙxⁿ + ... + a₁x + a₀. Domain: ℝ. Mulus, tidak putus.

2. Fungsi Rasional: f(x) = P(x)/Q(x). Domain: Q(x) ≠ 0. Bisa punya asimtot vertikal.

3. Fungsi Eksponen: f(x) = aˣ, a > 0. Domain: ℝ. Range: (0, ∞). Tumbuh/peluruhan eksponensial.

4. Fungsi Logaritma: f(x) = ᵃlog x. Domain: (0, ∞). Range: ℝ. Invers dari eksponen.

5. Fungsi Trigonometri: sin x, cos x, tan x, dll. Periodik. Penting untuk gelombang dan osilasi.

6. Fungsi Nilai Mutlak: f(x) = |x|. Grafik V. Tidak mulus di x = 0.

7. Fungsi Piecewise: Didefinisikan dengan aturan berbeda pada interval berbeda.

1.2 Representasi Fungsi — Empat Cara

1. Verbal: "Fungsi kuadrat dari x adalah x dikuadratkan lalu ditambah 1."

2. Aljabar (Rumus): f(x) = x² + 1.

3. Tabel: x=0→1, x=1→2, x=2→5, x=3→10.

4. Grafik: Kurva parabola pada bidang Kartesius.

Keempat cara ini mendeskripsikan fungsi yang sama. Pilih yang paling sesuai dengan konteks.

1.3 Operasi pada Fungsi

(f + g)(x) = f(x) + g(x) — penjumlahan.
(f - g)(x) = f(x) - g(x) — pengurangan.
(f · g)(x) = f(x) · g(x) — perkalian.
(f / g)(x) = f(x) / g(x) — pembagian, g(x) ≠ 0.

(f ∘ g)(x) = f(g(x)) — komposisi. Kerjakan g dulu, hasilnya masukkan ke f.

f⁻¹(x) — fungsi invers. Membalikkan f: jika f(a) = b, maka f⁻¹(b) = a.

1.4 Simetri Fungsi — Ganjil dan Genap

Fungsi Genap: f(-x) = f(x). Grafik simetris terhadap sumbu Y. Contoh: f(x) = x², cos x, |x|.

Fungsi Ganjil: f(-x) = -f(x). Grafik simetris terhadap titik asal. Contoh: f(x) = x³, sin x, x|x|.

Bukan keduanya: Kebanyakan fungsi tidak ganjil maupun genap. Contoh: f(x) = eˣ, x² + x.

2. Limit — Inti dari Kalkulus

Limit adalah pertanyaan: "Ke mana nilai f(x) mendekati ketika x mendekati suatu nilai c, tapi tidak harus tepat di c?"

Notasi: lim[x→c] f(x) = L

Dibaca: "Limit dari f(x) ketika x mendekati c adalah L."

Artinya: Ketika x semakin dekat ke c (dari kiri maupun kanan), nilai f(x) semakin dekat ke L.

Mengapa kita peduli limit?

Bayangkan fungsi f(x) = (x² - 1)/(x - 1).

Coba hitung f(1): (1 - 1)/(1 - 1) = 0/0 → tidak terdefinisi!

Tapi coba hitung untuk nilai yang dekat ke 1:
x = 0,9 → f(0,9) = (0,81-1)/(-0,1) = (-0,19)/(-0,1) = 1,9
x = 0,99 → f(0,99) = (0,9801-1)/(-0,01) = 1,99
x = 0,999 → f(0,999) = 1,999
x = 1,001 → f(1,001) = 2,001
x = 1,01 → f(1,01) = 2,01

Jelas: f(x) mendekati 2 saat x mendekati 1, meskipun f(1) tidak terdefinisi!

Jadi, lim[x→1] (x²-1)/(x-1) = 2.

Inilah kekuatan limit: limit tidak peduli dengan nilai fungsi di titik itu. Limit hanya peduli dengan perilaku fungsi di sekitar titik itu. Ini memungkinkan kita menangani kasus 0/0, ∞/∞, dan bentuk tak tentu lainnya.

2.1 Menghitung Limit — Metode Substitusi Langsung

Langkah pertama selalu: substitusi langsung x = c ke f(x).

Jika hasilnya terdefinisi dan bukan bentuk tak tentu → itu adalah limitnya.

Jika hasilnya 0/0, ∞/∞, 0×∞, ∞-∞, 1^∞, 0⁰, ∞⁰ → perlu manipulasi aljabar.
lim[x→2] (x² + 3x) = 4 + 6 = 10 ✅ (substitusi langsung berhasil)

lim[x→3] 1/(x-3) → 1/0 → tidak ada (bukan bentuk tak tentu, limit menuju ±∞)

lim[x→1] (x²-1)/(x-1) → 0/0 → bentuk tak tentu, perlu manipulasi.

2.2 Menghitung Limit — Metode Pemfaktoran

lim[x→1] (x² - 1)/(x - 1)

Faktorkan: x² - 1 = (x - 1)(x + 1).

= lim[x→1] (x-1)(x+1)/(x-1)
= lim[x→1] (x + 1)   (coret (x-1), untuk x ≠ 1)
= 2 ✅
lim[x→3] (x² - 9)/(x - 3)

= lim[x→3] (x-3)(x+3)/(x-3)
= lim[x→3] (x + 3)
= 6 ✅

2.3 Menghitung Limit — Metode Mengalikan Sekawan

Untuk limit yang melibatkan akar, kalikan pembilang dan penyebut dengan sekawan-nya.

lim[x→0] (√(x+1) - 1)/x

Substitusi langsung: (1-1)/0 = 0/0 → bentuk tak tentu.

Kalikan dengan sekawan (√(x+1) + 1):

= lim[x→0] [(√(x+1)-1)(√(x+1)+1)] / [x(√(x+1)+1)]
= lim[x→0] [(x+1) - 1] / [x(√(x+1)+1)]
= lim[x→0] x / [x(√(x+1)+1)]
= lim[x→0] 1/(√(x+1)+1)
= 1/(√1 + 1) = 1/2 ✅

2.4 Menghitung Limit — Metode Pembagian dengan Pangkat Tertinggi

Untuk limit di tak hingga dari fungsi rasional, bagi pembilang dan penyebut dengan pangkat tertinggi x di penyebut.

lim[x→∞] (3x² + 2x)/(x² + 5)

Bagi pembilang dan penyebut dengan x² (pangkat tertinggi):

= lim[x→∞] (3 + 2/x)/(1 + 5/x²)

Saat x → ∞: 2/x → 0, 5/x² → 0.

= (3 + 0)/(1 + 0) = 3 ✅

2.5 Limit yang Tidak Ada (DNE — Does Not Exist)

Limit tidak ada jika:

1. Limit kiri ≠ limit kanan (fungsi "melompat").
2. Nilai fungsi menuju ±∞ (asimtot vertikal).
3. Fungsi berosilasi tak menentu (contoh: sin(1/x) saat x→0).

3. Limit Sepihak — Dari Kiri dan Dari Kanan

Limit kiri: lim[x→c⁻] f(x) = L — x mendekati c dari kiri (nilai x < c).

Limit kanan: lim[x→c⁺] f(x) = L — x mendekati c dari kanan (nilai x > c).

Limit dua sisi ada jika dan hanya jika limit kiri = limit kanan.

Mengapa limit sepihak penting? Karena banyak fungsi yang "berbeda" di kiri dan kanan suatu titik—terutama fungsi piecewise dan fungsi dengan nilai mutlak.

f(x) = { x², jika x < 1
        { 2x, jika x ≥ 1

lim[x→1⁻] f(x) = 1² = 1 (dari kiri, pakai x²).
lim[x→1⁺] f(x) = 2(1) = 2 (dari kanan, pakai 2x).

Karena 1 ≠ 2, maka lim[x→1] f(x) tidak ada. ❌
lim[x→0⁺] |x|/x

Untuk x > 0: |x| = x → |x|/x = 1.
Untuk x < 0: |x| = -x → |x|/x = -1.

lim[x→0⁺] |x|/x = 1 (limit kanan).
lim[x→0⁻] |x|/x = -1 (limit kiri).

Karena 1 ≠ -1, maka lim[x→0] |x|/x tidak ada. ❌
