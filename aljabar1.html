Part 1 • Ekspresi & Identitas Aljabar

📖 Pendahuluan

Selamat datang di Bab 1 Aljabar!

Di Dasar Matematika, kita selalu berurusan dengan angka konkret: 2 + 3, 7 × 8, √25. Sekarang kita naik level. Kita akan belajar bekerja dengan variabel—simbol yang mewakili bilangan yang belum diketahui.

Kenapa ini penting? Karena di dunia nyata, kita sering tidak tahu semua angkanya. Kita tahu "harga 3 buku dan 2 pensil adalah Rp50.000", tapi tidak tahu harga per bukunya. Dengan aljabar, kita bisa menyatakannya dalam simbol, lalu mencari nilainya secara sistematis.

Bab pertama ini adalah fondasi. Kita akan belajar:

  • Bagaimana melakukan operasi pada ekspresi aljabar.

  • Bagaimana memfaktorkan ekspresi aljabar.
  • Bagaimana menyederhanakan pecahan yang mengandung variabel.

Kalau kamu kuat di sini, seluruh perjalanan aljabar selanjutnya akan jauh lebih mudah.

1. Apa Itu Bentuk Aljabar?

Bentuk aljabar (atau ekspresi aljabar) adalah kombinasi dari variabel, angka, dan operasi matematika (+, -, ×, ÷, pangkat, akar) yang ditulis menjadi satu kesatuan.

3x + 2y - 7 4a² - 2ab + b² 2x³ + 5x² - x + 8 (x + 3)(x - 2)

Bagian-Bagian Bentuk Aljabar:

Variabel = simbol yang mewakili bilangan (biasanya huruf: x, y, a, b, n, t).
Koefisien = angka yang menempel pada variabel (pengali variabel).
Konstanta = angka yang berdiri sendiri, tidak menempel pada variabel.
Suku = bagian dari ekspresi yang dipisahkan oleh tanda + atau -.

Contoh: Membedah 5x² - 3x + 7

5x² → suku pertama. Variabel: x, pangkat: 2, koefisien: 5.
-3x → suku kedua. Variabel: x, pangkat: 1, koefisien: -3.
+7 → suku ketiga. Konstanta: 7 (tidak punya variabel).

Ekspresi ini punya 3 suku → disebut trinomial.

Jenis-Jenis Ekspresi Berdasarkan Jumlah Suku

Monomial = 1 suku. Contoh: 3x, -5a², 7.
Binomial = 2 suku. Contoh: x + 5, 2a - 3b, x² - 4.
Trinomial = 3 suku. Contoh: x² + 2x + 1, a² - 2ab + b².
Polinomial = banyak suku (umumnya lebih dari 1).

Suku Sejenis dan Tidak Sejenis

Suku sejenis adalah suku-suku yang memiliki variabel dan pangkat yang sama persis. Hanya koefisiennya yang boleh berbeda.

Sejenis: 3x dan 7x (variabel x, pangkat 1). 5a² dan -2a² (variabel a, pangkat 2).
Tidak sejenis: 3x dan 3y (variabel beda). 4x dan 4x² (pangkat beda). 2ab dan 2a²b (pangkat beda).

Kenapa ini penting? Karena hanya suku sejenis yang bisa dijumlahkan atau dikurangkan.

2. Operasi Aljabar — Penjumlahan dan Pengurangan

Aturan utamanya sederhana:

Hanya suku sejenis yang bisa dijumlahkan atau dikurangkan.
Caranya: jumlahkan atau kurangkan koefisiennya, variabelnya tetap.

Mengapa?

Bayangkan 3x + 5x. x adalah suatu bilangan (entah berapa). 3x artinya 3 kali x. 5x artinya 5 kali x. Totalnya: 3 kali x + 5 kali x = (3+5) kali x = 8x. Ini sama seperti: 3 apel + 5 apel = 8 apel.

Contoh Penjumlahan

3x + 5x = (3+5)x = 8x

4a² + 7a² = (4+7)a² = 11a²

2xy + 6xy - 3xy = (2+6-3)xy = 5xy

(3x + 2y) + (5x - y) = 3x + 2y + 5x - y = (3+5)x + (2-1)y = 8x + y

Contoh Pengurangan

7x - 2x = (7-2)x = 5x

(5a + 3b) - (2a + b) = 5a + 3b - 2a - b = (5-2)a + (3-1)b = 3a + 2b

(4x² - 3x + 1) - (x² + 2x - 5) = 4x² - 3x + 1 - x² - 2x + 5 = 3x² - 5x + 6

Perhatikan tanda kurung! Saat mengurangkan ekspresi dalam kurung, tanda di dalamnya berubah: -(a + b) = -a - b.

💡 Tips: Saat mengurangkan bentuk aljabar, buka kurung dulu, baru gabungkan suku sejenis. Kesalahan paling umum adalah lupa mengubah tanda setelah tanda minus.

3. Operasi Aljabar — Perkalian

Perkalian dalam aljabar mengikuti sifat distributif: setiap suku di faktor pertama dikalikan dengan setiap suku di faktor kedua.

Perkalian Monomial dengan Monomial

Kalikan koefisien dengan koefisien, variabel dengan variabel.
Gunakan sifat pangkat: xᵃ × xᵇ = xᵃ⁺ᵇ
3x × 5x = (3×5) × (x×x) = 15x²

2a² × 4a³ = (2×4) × a²⁺³ = 8a⁵

(-3xy)(2x²y) = -6x³y²

Perkalian Monomial dengan Binomial/Polinomial

Gunakan sifat distributif: a(b + c) = ab + ac

2x(3x + 5) = 2x·3x + 2x·5 = 6x² + 10x

-3a(2a² - a + 4) = -6a³ + 3a² - 12a

Perkalian Binomial dengan Binomial — Metode FOIL

FOIL adalah singkatan dari First, Outer, Inner, Last. Ini adalah cara sistematis mengalikan dua binomial.

(a + b)(c + d) = First + Outer + Inner + Last
= a·c + a·d + b·c + b·d

Contoh: (x + 3)(x + 2)

First: x × x = x²
Outer: x × 2 = 2x
Inner: 3 × x = 3x
Last: 3 × 2 = 6

(x + 3)(x + 2) = x² + 2x + 3x + 6 = x² + 5x + 6 ✅

Mengapa FOIL bekerja? FOIL adalah penerapan sifat distributif dua kali:

(x + 3)(x + 2) = (x + 3)·x + (x + 3)·2
= x·x + 3·x + x·2 + 3·2
= x² + 3x + 2x + 6
= x² + 5x + 6

Jadi FOIL bukan sihir, cuma cara cepat dari distributif biasa.

Contoh Lain Perkalian Binomial

(2x - 1)(x + 4) = 2x·x + 2x·4 + (-1)·x + (-1)·4
= 2x² + 8x - x - 4
= 2x² + 7x - 4

(a + b)(a - b) = a·a + a·(-b) + b·a + b·(-b)
= a² - ab + ab - b²
= a² - b² ✅ (Ini adalah identitas selisih kuadrat!)

Perkalian Polinomial dengan Polinomial

Prinsipnya sama: setiap suku di polinomial pertama dikalikan dengan setiap suku di polinomial kedua, lalu gabungkan suku sejenis.

(x² + 2x + 1)(x + 3)

= x²(x+3) + 2x(x+3) + 1(x+3)
= x³ + 3x² + 2x² + 6x + x + 3
= x³ + 5x² + 7x + 3 ✅

4. Operasi Aljabar — Pembagian

Pembagian aljabar pada dasarnya adalah membagi koefisien dan mengurangkan pangkat variabel yang sama.

xᵃ ÷ xᵇ = xᵃ⁻ᵇ (x ≠ 0)

Pembagian Monomial oleh Monomial

12x⁵ ÷ 3x² = (12÷3) × x⁵⁻² = 4x³

20a³b² ÷ 5ab = (20÷5) × a³⁻¹ × b²⁻¹ = 4a²b

-18x⁴ ÷ 6x = -3x³

Pembagian Polinomial oleh Monomial

Bagi setiap suku di polinomial dengan monomialnya.

(6x³ + 9x² - 3x) ÷ 3x
= 6x³÷3x + 9x²÷3x - 3x÷3x
= 2x² + 3x - 1 ✅

Verifikasi: 3x(2x² + 3x - 1) = 6x³ + 9x² - 3x. Cocok!

Pembagian Polinomial oleh Polinomial (Pembagian Panjang)

Untuk pembagian yang lebih kompleks, kita gunakan pembagian panjang, mirip seperti pembagian bilangan biasa.

Contoh: (x² + 5x + 6) ÷ (x + 2)

Langkah 1: x² ÷ x = x. Tulis x di hasil.
Langkah 2: x(x + 2) = x² + 2x. Kurangkan.
(x² + 5x + 6) - (x² + 2x) = 3x + 6

Langkah 3: 3x ÷ x = 3. Tulis 3 di hasil.
Langkah 4: 3(x + 2) = 3x + 6. Kurangkan.
(3x + 6) - (3x + 6) = 0

Hasil: x + 3 ✅ (sisa 0)

Verifikasi: (x + 2)(x + 3) = x² + 5x + 6. Cocok!

💡 Tips: Saat melakukan pembagian panjang aljabar:
  1. Urutkan polinomial dari pangkat tertinggi ke terendah.
  2. Kalau ada pangkat yang loncat (misal x³ + 1, tanpa x² dan x), isi dengan 0x² + 0x.
  3. Selalu verifikasi dengan mengalikan hasil × pembagi + sisa.

5. Identitas Aljabar — Pola yang Selalu Benar

Identitas aljabar adalah persamaan yang selalu benar, tidak peduli berapa pun nilai variabelnya. Berbeda dengan persamaan biasa yang hanya benar untuk nilai tertentu (misal 2x = 6 hanya benar untuk x = 3), identitas aljabar benar untuk semua nilai variabel.

Mengapa identitas penting?

Identitas adalah "pola pintas" yang menyederhanakan perhitungan. Daripada mengalikan (x+3)(x+3) secara manual setiap kali, kita bisa langsung pakai identitas (a+b)² = a² + 2ab + b².

Identitas 1: Kuadrat Binomial — (a + b)²

(a + b)² = a² + 2ab + b²

Pembuktian:

(a + b)² = (a + b)(a + b)
= a·a + a·b + b·a + b·b   (FOIL)
= a² + ab + ab + b²
= a² + 2ab + b² ✅

Jangan salah umum: (a + b)² bukan a² + b²! Suku tengah 2ab sering dilupakan.

(x + 5)² = x² + 2·x·5 + 5² = x² + 10x + 25

(2a + 3)² = (2a)² + 2·(2a)·3 + 3² = 4a² + 12a + 9

(3x + 4y)² = (3x)² + 2·(3x)(4y) + (4y)² = 9x² + 24xy + 16y²

Identitas 2: Kuadrat Binomial — (a - b)²

(a - b)² = a² - 2ab + b²

Pembuktian:

(a - b)² = (a - b)(a - b)
= a·a + a·(-b) + (-b)·a + (-b)(-b)
= a² - ab - ab + b²
= a² - 2ab + b² ✅

Atau bisa juga: (a - b)² = (a + (-b))² = a² + 2a(-b) + (-b)² = a² - 2ab + b².

(x - 7)² = x² - 2·x·7 + 7² = x² - 14x + 49

(3x - 2)² = (3x)² - 2·(3x)·2 + 2² = 9x² - 12x + 4

Identitas 3: Selisih Kuadrat — (a + b)(a - b)

(a + b)(a - b) = a² - b²

Ini adalah identitas paling penting karena sangat sering dipakai, terutama dalam faktorisasi.

Pembuktian:

(a + b)(a - b) = a·a + a·(-b) + b·a + b·(-b)
= a² - ab + ab - b²
= a² - b² ✅

Suku tengah (-ab + ab) saling menghilangkan. Inilah kenapa hasilnya a² - b².

(x + 3)(x - 3) = x² - 3² = x² - 9

(2x + 5)(2x - 5) = (2x)² - 5² = 4x² - 25

(3a + 4b)(3a - 4b) = (3a)² - (4b)² = 9a² - 16b²

Identitas 4: (a + b)(a + c) — Suku Depan Sama

(x + a)(x + b) = x² + (a + b)x + ab

Ini generalisasi dari FOIL ketika suku pertama kedua binomial sama (sama-sama x).

Pembuktian:

(x + a)(x + b) = x·x + x·b + a·x + a·b
= x² + bx + ax + ab
= x² + (a + b)x + ab ✅

Pola cepatnya: suku tengah = jumlah a dan b, suku terakhir = hasil kali a dan b.

(x + 2)(x + 5) = x² + (2+5)x + (2×5) = x² + 7x + 10

(x + 3)(x + 8) = x² + (3+8)x + (3×8) = x² + 11x + 24

(x - 2)(x + 7) = x² + (-2+7)x + (-2×7) = x² + 5x - 14

(x - 4)(x - 3) = x² + (-4-3)x + (-4)(-3) = x² - 7x + 12

Identitas 5: Kubus Binomial — (a + b)³ dan (a - b)³

(a + b)³ = a³ + 3a²b + 3ab² + b³

(a - b)³ = a³ - 3a²b + 3ab² - b³

Pembuktian (a + b)³:

(a + b)³ = (a + b)(a + b)²
= (a + b)(a² + 2ab + b²)
= a(a² + 2ab + b²) + b(a² + 2ab + b²)
= a³ + 2a²b + ab² + a²b + 2ab² + b³
= a³ + 3a²b + 3ab² + b³ ✅
(x + 2)³ = x³ + 3·x²·2 + 3·x·2² + 2³ = x³ + 6x² + 12x + 8

(y - 3)³ = y³ - 3·y²·3 + 3·y·3² - 3³ = y³ - 9y² + 27y - 27

Identitas 6: Jumlah dan Selisih Kubus

a³ + b³ = (a + b)(a² - ab + b²)

a³ - b³ = (a - b)(a² + ab + b²)

Pembuktian a³ + b³:

(a + b)(a² - ab + b²)
= a(a² - ab + b²) + b(a² - ab + b²)
= a³ - a²b + ab² + a²b - ab² + b³
= a³ + (-a²b + a²b) + (ab² - ab²) + b³
= a³ + b³ ✅

Semua suku tengah saling menghilangkan!

x³ + 8 = x³ + 2³ = (x + 2)(x² - 2x + 4)

27a³ - 1 = (3a)³ - 1³ = (3a - 1)(9a² + 3a + 1)

6. Faktorisasi — Mengembalikan ke Bentuk Perkalian

Faktorisasi adalah proses mengubah suatu ekspresi aljabar menjadi perkalian faktor-faktor yang lebih sederhana.

Ini adalah kebalikan dari perkalian. Kalau (x+2)(x+3) dikalikan menjadi x²+5x+6, maka memfaktorkan x²+5x+6 menghasilkan (x+2)(x+3).

Mengapa faktorisasi penting?

  • Menyederhanakan pecahan aljabar.
  • Menyelesaikan persamaan kuadrat.
  • Mencari akar polinomial.
  • Menyederhanakan perhitungan.

Metode 1: Faktor Persekutuan Terbesar

Cari faktor yang sama di semua suku, keluarkan.

6x² + 9x = 3x(2x + 3)   (FPB dari 6x² dan 9x adalah 3x)

4a³ - 8a² + 12a = 4a(a² - 2a + 3)

5x²y + 10xy² = 5xy(x + 2y)

Verifikasi: kalikan kembali dengan distributif, harus kembali ke bentuk awal.

Metode 2: Selisih Kuadrat — a² - b² = (a+b)(a-b)

Ini adalah kebalikan dari identitas selisih kuadrat.

x² - 25 = x² - 5² = (x + 5)(x - 5)

4x² - 9 = (2x)² - 3² = (2x + 3)(2x - 3)

49a² - 16b² = (7a)² - (4b)² = (7a + 4b)(7a - 4b)

x⁴ - 16 = (x²)² - 4² = (x² + 4)(x² - 4) = (x² + 4)(x + 2)(x - 2)

Perhatikan contoh terakhir: setelah memfaktorkan sekali, x² - 4 masih bisa difaktorkan lagi! Selalu cek apakah hasilnya masih bisa difaktorkan.

Metode 3: Trinomial Bentuk x² + bx + c

Cari dua bilangan yang:

Jumlahnya = b (koefisien x)
Hasil kalinya = c (konstanta)

Contoh: Faktorkan x² + 7x + 12

Cari dua bilangan yang jumlahnya 7 dan hasil kalinya 12.

Pasangan faktor 12: (1,12), (2,6), (3,4).
Yang jumlahnya 7: 3 dan 4.

Jadi, x² + 7x + 12 = (x + 3)(x + 4) ✅

Verifikasi: (x+3)(x+4) = x² + 4x + 3x + 12 = x² + 7x + 12. Cocok!

Contoh lain:

x² + 5x + 6 → cari jumlah 5, kali 6 → (x+2)(x+3)

x² - 5x + 6 → cari jumlah -5, kali 6 → (x-2)(x-3)

x² + x - 6 → cari jumlah 1, kali -6 → (x+3)(x-2)

x² - x - 6 → cari jumlah -1, kali -6 → (x-3)(x+2)

Mengapa ini bekerja? Dari identitas (x+p)(x+q) = x² + (p+q)x + pq. Jadi b = p+q dan c = pq. Kita tinggal mencari p dan q.

Metode 4: Trinomial Bentuk ax² + bx + c (a ≠ 1)

Untuk trinomial dengan koefisien x² bukan 1, kita perlu modifikasi sedikit.

Metode AC:

  1. Hitung a × c.
  2. Cari dua bilangan yang jumlahnya = b dan hasil kalinya = a × c.
  3. Pecah suku tengah bx menggunakan dua bilangan itu.
  4. Faktorkan dengan pengelompokan.

Contoh: Faktorkan 2x² + 7x + 3

a = 2, b = 7, c = 3
a × c = 6

Cari dua bilangan: jumlah = 7, hasil kali = 6
→ 6 dan 1

Pecah 7x menjadi 6x + x:
2x² + 7x + 3 = 2x² + 6x + x + 3

Kelompokkan:
= (2x² + 6x) + (x + 3)
= 2x(x + 3) + 1(x + 3)
= (2x + 1)(x + 3) ✅

Verifikasi: (2x+1)(x+3) = 2x² + 6x + x + 3 = 2x² + 7x + 3. Cocok!

Contoh lain:

3x² + 11x + 6 → a×c=18, jumlah 11 → 9 dan 2
= 3x² + 9x + 2x + 6 = 3x(x+3) + 2(x+3) = (3x+2)(x+3)

6x² + x - 12 → a×c=-72, jumlah 1 → 9 dan -8
= 6x² + 9x - 8x - 12 = 3x(2x+3) - 4(2x+3) = (3x-4)(2x+3)

Metode 5: Kuadrat Sempurna

Kenali pola a² ± 2ab + b².

x² + 6x + 9 = x² + 2·x·3 + 3² = (x + 3)²

x² - 10x + 25 = x² - 2·x·5 + 5² = (x - 5)²

4x² + 12x + 9 = (2x)² + 2·(2x)·3 + 3² = (2x + 3)²

Metode 6: Jumlah/Selisih Kubus

x³ + 27 = x³ + 3³ = (x + 3)(x² - 3x + 9)

8a³ - 125 = (2a)³ - 5³ = (2a - 5)(4a² + 10a + 25)

7. Pecahan Aljabar — Pecahan yang Mengandung Variabel

Pecahan aljabar adalah pecahan yang pembilang atau penyebutnya mengandung variabel.

3/x    (x+2)/(x-1)    (x²-4)/(x+2)

Aturan mainnya sama persis seperti pecahan biasa di Dasar 3. Bedanya, sekarang pembilang dan penyebutnya adalah ekspresi aljabar.

Menyederhanakan Pecahan Aljabar

Langkah-langkah:

  1. Faktorkan pembilang dan penyebut.
  2. Coret faktor yang sama (asalkan ≠ 0).
Sederhanakan: (x² - 4) / (x + 2)

Pembilang: x² - 4 = (x+2)(x-2)
Penyebut: x + 2

= (x+2)(x-2) / (x+2)
= x - 2   (untuk x ≠ -2) ✅

Kenapa harus x ≠ -2? Karena kalau x = -2, penyebut aslinya nol (tidak terdefinisi). Meskipun bentuk sederhananya jadi x-2, domain awalnya tetap tidak boleh x = -2.

(x² + 5x + 6) / (x² + 3x + 2)

Pembilang: (x+2)(x+3)
Penyebut: (x+2)(x+1)

= (x+2)(x+3) / (x+2)(x+1)
= (x+3) / (x+1)   (untuk x ≠ -2, x ≠ -1) ✅

Penjumlahan dan Pengurangan Pecahan Aljabar

Prinsipnya sama seperti pecahan biasa: samakan penyebut dulu.

2/x + 3/x = (2+3)/x = 5/x

1/(x+1) + 1/(x-1)
= (x-1)/(x+1)(x-1) + (x+1)/(x+1)(x-1)
= (x-1 + x+1) / (x²-1)
= 2x / (x² - 1) ✅

Contoh dengan penyebut berbeda:

3/(x+2) - 1/(x-1)

KPK penyebut: (x+2)(x-1)

= 3(x-1)/(x+2)(x-1) - 1(x+2)/(x+2)(x-1)
= [3(x-1) - (x+2)] / (x+2)(x-1)
= (3x - 3 - x - 2) / (x+2)(x-1)
= (2x - 5) / (x+2)(x-1) ✅

Perkalian Pecahan Aljabar

(a/b) × (c/d) = (a × c) / (b × d)
(x/x+1) × (x+1/x²) = x(x+1) / (x+1)x² = 1/x   (x ≠ 0, x ≠ -1)

(x²-1)/(x+2) × (x+2)/(x-1) = (x²-1)/(x-1) = (x+1)(x-1)/(x-1) = x+1

Pembagian Pecahan Aljabar

(a/b) ÷ (c/d) = (a/b) × (d/c) = (a × d) / (b × c)
(x/x-2) ÷ (x²/x²-4)
= (x/x-2) × (x²-4/x²)
= x(x²-4) / (x-2)x²
= x(x+2)(x-2) / (x-2)x²
= (x+2) / x   (x ≠ 0, x ≠ 2, x ≠ -2) ✅

📝 Ringkasan Cepat

  • Bentuk aljabar: kombinasi variabel, koefisien, konstanta, operasi.
  • Suku sejenis: variabel dan pangkat sama → bisa dijumlah/kurangi.
  • Perkalian: distributif, FOIL untuk binomial.
  • Pembagian: kurangkan pangkat variabel yang sama.
  • (a+b)² = a² + 2ab + b² — jangan lupa 2ab!
  • (a+b)(a-b) = a² - b² — identitas paling sering dipakai.
  • Faktorisasi: FPB, selisih kuadrat, trinomial, kuadrat sempurna.
  • Pecahan aljabar: faktorkan → coret → sederhanakan.

✨ Tips & Trik

  • Selalu cek tanda. Kesalahan paling umum adalah salah tanda negatif, terutama saat mengurangkan bentuk aljabar.
  • Hafalkan identitas dasar. (a+b)², (a-b)², dan (a+b)(a-b). Tiga ini akan terus muncul di semua materi aljabar selanjutnya.
  • Verifikasi faktorisasi. Setelah memfaktorkan, kalikan kembali. Kalau hasilnya kembali ke bentuk awal, pemfaktoranmu benar.
  • Jangan lupa domain. Saat menyederhanakan pecahan aljabar, catat nilai x yang membuat penyebut asli = 0.
  • Latih mata melihat pola. Semakin sering latihan, kamu akan semakin cepat mengenali selisih kuadrat, kuadrat sempurna, atau trinomial yang bisa difaktorkan.
